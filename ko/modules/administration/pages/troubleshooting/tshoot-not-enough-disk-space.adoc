= 디스크 공간 부족 문제 해결
:description: 성공적인 마이그레이션을 위해 디스크 공간을 구성하려면 사용 가능한 디스크 공간을 확인하고, 중요한 파일을 별도의 XFS 파일 시스템으로 이동하고, fstab을 편집하십시오.

마이그레이션을 시작하기 전에 사용 가능한 디스크 공간을 확인하십시오.
`/var/spacewalk`과 `/var/lib/pgsql`는 별도의 XFS 파일 시스템에 배치하는 것을 권장합니다.

별도의 파일 시스템을 설정할 때는 `/etc/fstab`을(를) 편집하고 `/var/lib/pqsql` 하위 볼륨을 제거합니다.
변경 사항을 적용하려면 서버를 재부팅하십시오.

업그레이드 문제에 대한 더 많은 정보를 얻으려면 마이그레이션 로그 파일을 확인하십시오.
로그 파일은 업그레이드 중인 시스템의 `/var/log/rhn/migration.log`에 위치합니다.
