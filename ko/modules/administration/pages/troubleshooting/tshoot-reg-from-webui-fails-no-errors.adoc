= {webui}에서 등록 문제 해결을 시도하면 실패하며, 오류가 표시되지 않습니다.
:description: Web UI에서 등록 문제 해결 방법을 알아보고, 오류 메시지를 확인하기 위해 /var/log/salt-ssh.log의 로컬 로그를 검사하십시오.
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀

:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{webui}에서의 초기 등록을 위해 모든 Salt 클라이언트는 Salt SSH를 사용합니다.

기본적으로 Salt SSH 클라이언트는 서버에 오류를 다시 보고하지 않습니다.

그러나 Salt SSH 클라이언트는 `/var/log/salt-ssh.log`에 로그를 로컬로 저장하며, 해당 로그에서 오류를 직접 확인할 수 있습니다.
