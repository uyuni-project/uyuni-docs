일부 업그레이드에서는 여러 개의 `*.rpmnew` 및 `*.rpmsave` 파일이 생성될 수 있습니다. 이러한 파일의 존재 여부나 개수가 수동 작업이 필요함을 나타내는 것은 아닙니다. SUSE Multi-Linux Manager 컨테이너 환경에서는 업그레이드 프로세스 중에 필요한 구성 변경 사항이 자동으로 적용됩니다. 이러한 파일은 패키지된 기본값과 기존 구성 파일 간의 차이로 인해 생성되며, 병합할 의도가 없는 내부 또는 정보 제공용 변경 사항이 포함될 수도 있습니다. 구성 변경을 위해 수동 작업이 필요한 경우, 릴리스 정보 또는 업그레이드 문서에 명시적으로 기록됩니다.

이러한 파일을 업그레이드 후 체크리스트로 취급하거나 일괄적으로 병합하지 마십시오. 특정 문제를 적극적으로 해결 중이고 해당 파일에 포함된 구성의 영향을 이해하고 있는 경우에만 파일을 검토하십시오. 변경이 필요한 경우, 이러한 파일의 차이점을 복사하는 방식이 아니라 알려진 요구 사항에 따라 의도적으로 적용해야 합니다. 확실하지 않은 경우 파일을 변경하지 마십시오.
