== 프록시용 활성화 키 생성


.절차 활성화 키 생성
[role=procedure]
_____

. menu:시스템[활성화 키]로 이동한 후, btn:[키 생성]을 클릭합니다.

. 프록시 호스트에 대해 {sl-micro} {microversion} 또는 {sles} {bci-mlm}를 부모 채널로 하여 활성화 키를 생성합니다.
  이 키에는 모든 권장 채널과 프록시를 확장 하위 채널로 포함해야 합니다.

. 프록시 호스트를 `default` 클라이언트로 부트스트래핑합니다.

_____

[WARNING]
====
프록시가 원래 공급업체 채널에만 할당되도록 합니다.

복제된 채널 할당은 현재 지원되지 않습니다.
====
