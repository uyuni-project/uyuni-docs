= 데이터베이스 백업 볼륨
:revdate: 2025-07-29
:page-revdate: {revdate}

`mgradm migration` 또는 `mgradm upgrade`를 사용한 서버 마이그레이션이나 업그레이드는 데이터베이스 백업이 포함된 볼륨을 생성할 수 있습니다.

{postgresql} 데이터베이스 버전이 증가할 때, 이전 데이터베이스는 업그레이드를 실행하기 전에 별도의 위치에 저장해야 합니다.
이 목적을 위해 `mgradm`는 동적으로 볼륨 `var-pgsql-backup`을 생성합니다.
마이그레이션 또는 업그레이드가 완료되고 사용자가 새로운 시스템이 예상대로 작동하는지 확인하면, 이 볼륨은 안전하게 제거할 수 있습니다.
