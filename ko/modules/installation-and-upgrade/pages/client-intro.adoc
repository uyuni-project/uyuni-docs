[[client-upgrade]]
= 클라이언트 업그레이드
:description: YaST 도구를 사용하여 클라이언트 업그레이드를 관리하는 방법을 배우고, SUSE Multi-Linux Manager 웹 UI를 사용하여 최적의 기능을 위해 버전 관리를 구성하는 방법을 알아보세요.
:revdate: 2024-11-19
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


클라이언트는 자체 기본 운영 체제의 버전 관리 시스템을 사용합니다.
{suse} 운영 체제를 사용하는 클라이언트의 경우, {productname} {webui} 내에서 업그레이드를 수행할 수 있습니다.

클라이언트 업그레이드에 대한 자세한 정보는 xref:client-configuration:client-upgrades.adoc[]을(를) 참조하십시오.
