= 컨테이너 시작 및 중지
:description: 간단한 터미널 명령을 사용하여 SUSE Multi-Linux Manager 5.1 서버 컨테이너를 관리하고 재시작하십시오.
:revdate: 2024-02-13
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

다음 명령을 사용하여 {productname} {productnumber} 서버 컨테이너를 재시작, 시작 및 중지할 수 있습니다:


`restart` {productname} {productnumber} 서버를 다음 명령으로 실행하십시오:
----
# mgradm restart
5:23PM INF Welcome to mgradm
5:23PM INF Executing command: restart
----

서버를 `start`하시려면 다음 명령을 실행하십시오:
----
# mgradm start
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: start
----

서버를 `stop`하시려면 다음 명령을 실행하십시오:
----
# mgradm stop
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: stop
----
