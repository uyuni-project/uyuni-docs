= 시스템 유형
:description: 기본 시스템 유형을 할당하고 가상화 또는 컨테이너 빌드 호스트 기능을 추가하여 클라이언트 시스템을 구성하십시오.
:revdate: 2023-11-23
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

클라이언트는 시스템 유형에 따라 분류됩니다.
모든 클라이언트는 기본 시스템 유형과 추가 기능 시스템 유형을 모두 가질 수 있습니다.

모든 클라이언트의 기본 시스템 유형은 `Salt`입니다.

추가 기능 시스템 유형에는 가상 호스트로 작동하는 클라이언트를 위한 `Virtualization Host`와 빌드 호스트로 작동하는 클라이언트를 위한 `Container Build Host`가 포함됩니다.

menu:시스템[시스템 목록> 시스템 유형]으로 이동하여 추가 기능 시스템 유형을 조정할 수 있습니다.
추가 기능 시스템 유형을 변경할 클라이언트를 선택하고 `Add-On System Type`를 선택한 후 btn:[시스템 유형 추가] 또는 btn:[시스템 유형 제거]를 클릭하십시오.
