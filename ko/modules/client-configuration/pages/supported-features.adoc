= 지원되는 클라이언트 및 기능
:description: SUSE Linux Enterprise Server for SAP, AlmaLinux, CentOS, Debian, Oracle Linux, Red Hat을 포함한 다양한 Linux 운영 체제를 구성하고 관리할 수 있습니다.
:revdate: 2026-05-21
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}은 다양한 클라이언트 기술과 호환됩니다.
{sle} 또는 다른 Linux 운영 체제를 실행하는 Salt 클라이언트를 설치할 수 있으며, 다양한 하드웨어 옵션이 있습니다.

이 섹션에는 지원되는 클라이언트 시스템의 요약이 포함되어 있습니다. 각 클라이언트에서 사용할 수 있는 기능의 자세한 목록은 다음 페이지를 참조하십시오.


[[supported-features-clients]]
== 지원되는 클라이언트 시스템


클라이언트 운영 체제는 운영 체제를 제공하는 조직이 지원합니다.
버전 및 SP 수준은 {productname}과 함께 지원되기 위해 일반 지원(정상 또는 LTSS) 하에 있어야 합니다.
지원되는 제품 버전에 대한 자세한 내용은 https://www.suse.com/lifecycle.를 참조하십시오.


지원되는 클라이언트 운영 체제는 이 표에 나와 있습니다. 표의 아이콘에는 다음과 같은 의미가 있습니다:

* 이 운영 체제를 실행 중인 {check} 클라이언트는 {suse}에서 지원됩니다.
* 이 운영 체제를 실행 중인 {cross} 클라이언트는 {suse}에서 지원하지 않습니다.
* {question} 클라이언트는 현재 검토 중이며, 이후에 지원될 수도 있고 지원되지 않을 수도 있습니다.

[IMPORTANT]
====
클라이언트에서 실행되는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다.
====

== 추가 도구 패키지

`spacewalk-utils` 및 `spacewalk-utils-extras` 패키지는 추가 서비스와 기능을 제공할 수 있습니다. 사용자에게는 사용하지 않음으로 지정된 도구 사용을 중단하고, 문서에서 권장하는 대안으로 전환할 것을 권장합니다.

ifeval::[{mlm-content} == true]
`spacewalk-utils`의 {productname} 패키지는 {suse}에 의해 완전히 지원되며 다음과 같은 도구를 포함합니다:
endif::[]

[[spacewalk-utils]]
[cols="1,1,1", options="header"]
.Spacewalk 유틸리티
|===

| 도구 이름
| 설명
| 사용하지 않음?

| `spacewalk-common-channels`
| {scclongform}에서 제공되지 않는 채널 추가
|

| `spacewalk-hostname-rename`
| {productname} 서버의 호스트 이름 변경
|

| `spacewalk-clone-by-date`
| 특정 날짜로 채널 복제. 이 도구는 사용하지 않음으로 지정되었으며, 사용자는 CLM API로 전환해야 합니다.
| {check}

| `spacewalk-sync-setup`
| ISS 마스터 및 슬레이브 조직 매핑 설정
| {check}

| `spacewalk-manage-channel-lifecycle`
| 채널 라이프사이클 관리. 이 도구는 사용하지 않음으로 지정되었으며, 사용자는 CLM API로 전환해야 합니다.
| {check}

|===
