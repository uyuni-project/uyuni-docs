= 지원되는 {sle-micro} 기능
:description: SLE micro 운영 체제에 대한 지원되는 기능의 포괄적인 목록을 문서를 통해 확인하십시오.
:revdate: 2024-07-18
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {sle-micro}은 {suse}에 의해 지원됩니다.
====

[IMPORTANT]
====
현재 {sle-micro}은(는) 일반 미니언(`default` 연락 방법)으로만 지원됩니다. 또한 Salt SSH 클라이언트(`salt-ssh` 연락 방법)로 관리하는 작업도 진행 중입니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능은 사용 가능합니다.
* {cross} 이 기능은 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후 사용 가능 여부가 결정될 수 있습니다.
