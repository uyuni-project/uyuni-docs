[[virt-vhm]]
= 가상 호스트 관리자
:description: 가상 호스트 관리자를 사용하여 가상화 클라이언트를 구성하고 SUSE Multi-Linux Manager 웹 UI에 표시된 데이터를 수집합니다.
:revdate: 2025-04-15
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

가상 호스트 관리자(VHM)는 다양한 클라이언트 유형에서 정보를 수집하는 데 사용됩니다.

VHMs는 개인 또는 공용 클라우드 인스턴스를 수집하고 이를 가상화 그룹으로 구성하는 데 사용할 수 있습니다.
이렇게 구성된 가상화 클라이언트로 Taskomatic은 클라이언트에 대한 데이터를 수집하여 {productname} {webui}에 표시합니다.
VHMs는 가상화 클라이언트에 대해 구독 매칭 기능을 사용할 수 있도록 합니다.

{productname} 서버에서 VHM을 생성하고 이를 사용하여 사용 가능한 공용 클라우드 인스턴스를 인벤토리할 수 있습니다.
VHM을 사용하여 {kube}로 생성된 클러스터를 관리할 수도 있습니다.

* Amazon Web Services와 함께 VHM을 사용하는 방법에 대한 자세한 내용은 xref:client-configuration:vhm-aws.adoc[]을 참조하십시오.
* Microsoft Azure와 함께 VHM을 사용하는 방법에 대한 자세한 내용은 xref:client-configuration:vhm-azure.adoc[]을 참조하십시오.
* Nutanix와 함께 VHM을 사용하는 방법에 대한 자세한 내용은 xref:client-configuration:vhm-nutanix.adoc[]을 참조하십시오.
* VMWare vSphere와 함께 VHM을 사용하는 방법에 대한 자세한 내용은 xref:client-configuration:vhm-vmware.adoc[]을 참조하십시오.
* 다른 호스트와 함께 VHM을 사용하는 방법에 대한 자세한 내용은 xref:client-configuration:vhm-file.adoc[]을 참조하십시오.
