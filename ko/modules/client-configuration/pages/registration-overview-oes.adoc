= {oes} 클라이언트 등록
:description: SUSE Multi-Linux Manager 서버를 사용하여 Open Enterprise Server 클라이언트를 등록하는 방법을 배우고, 클라이언트 등록 방법에 대한 이해로 시작하십시오.
:revdate: 2026-05-05
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{oes} 클라이언트를 {productname} 서버에 등록할 수 있습니다.
방법과 자세한 내용은 클라이언트의 운영 체제에 따라 다릅니다.

시작하기 전에 클라이언트의 날짜 및 시간이 {productname} 서버와 정확히 동기화되었는지 확인하십시오.

활성화 키도 생성해야 합니다.

활성화 키 생성에 대한 자세한 내용은 xref:client-configuration:activation-keys.adoc[]을 참조하십시오.
