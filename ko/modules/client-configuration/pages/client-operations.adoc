= 클라이언트 작업
:description: SUSE Multi-Linux Manager 클라이언트 또는 클라이언트 그룹을 시스템 세트 관리자(System Set Manager)를 사용하여 관리하고, 구성을 통해 고급 클라이언트 옵션을 설정합니다.
:revdate: 2024-03-05
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

등록, 업그레이드, 소프트웨어 설치 또는 클라이언트 삭제 외에도 다른 작업을 수행할 수 있습니다. {productname} 클라이언트는 개별적으로 또는 클라이언트 세트로 관리할 수 있습니다.
클라이언트 세트는 시스템 세트 관리자(System Set Manager)를 사용하여 일회성 작업을 위해 그룹화하거나 시스템 그룹(System Groups)을 통해 영구적으로 그룹화할 수 있습니다.

고급 클라이언트 구성 옵션에 대해 일반 구성 관리 기능을 사용할 수 있습니다.
{productname} {webui}를 사용하여 사용자 정의 시스템 정보를 얻고, 클라이언트를 켜거나 끄고, 재부팅할 수 있습니다.

다음 섹션에서는 이러한 각 작업에 대한 자세한 설명을 제공합니다.
