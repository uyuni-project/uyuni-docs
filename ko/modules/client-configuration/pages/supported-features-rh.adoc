[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {rhel}는 {redhat}에 의해 지원됩니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 기능을 사용할 수 있습니다.
* {cross}이 기능을 사용할 수 없습니다.
* {question} 기능은 현재 검토 중이며, 향후 사용 여부가 결정될 수 있습니다.
