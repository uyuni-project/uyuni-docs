= 지원되는 {rocky} 기능
:description: 여러 클라이언트 전반에 걸쳐 다양한 Rocky Linux 버전의 사용 가능한 기능들을 현재 사용 여부, 지원 상태 및 잠재력 측면에서 비교합니다.
:revdate: 2024-09-30
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


이 표는 {rocky} 클라이언트에서 다양한 기능의 사용 가능 여부를 정리한 것입니다.


[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {rocky}은 {rocky} 커뮤니티에 의해 지원됩니다.
====


이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능은 사용 가능합니다.
* {cross} 이 기능은 사용할 수 없습니다.
* {question} 기능은 현재 검토 중이며, 향후 사용 여부가 결정될 수 있습니다.
