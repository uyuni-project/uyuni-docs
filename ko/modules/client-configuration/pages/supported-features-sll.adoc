= 지원되는 {sll} 기능
:description: SUSE Liberty Linux 클라이언트에서 시스템 패키지, 등록을 포함한 지원되는 기능의 포괄적인 목록을 확인하십시오.
:revdate: 2026-04-20
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

이 표는 {sll} 클라이언트에서 다양한 기능의 사용 가능 여부를 정리한 것입니다.

[NOTE]
====
{sll} 클라이언트는 때때로 {sleses}(SLES ES), Liberty, RES 또는 {redhat} {es}라고도 합니다.
====

[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {sll}은 {suse}에 의해 지원됩니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능은 사용 가능합니다.
* {cross} 이 기능은 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후 사용 가능 여부가 결정될 수 있습니다.
