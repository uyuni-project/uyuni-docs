= 지원되는 {openeuler} 기능
:description: 시스템 패키지, 등록 옵션, 패키지 설치를 포함하여 openEuler 클라이언트에서 지원되는 기능에 대해 알아보세요.
:revdate: 2026-06-09
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager

이 표는 {openeuler} 클라이언트에서 다양한 기능의 사용 가능 여부를 정리한 것입니다.

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
{productname}에서 {openeuler}에 대한 지원이 기술 미리보기로 제공됩니다.
====

endif::[]



[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {openeuler}은 {openeuler} 커뮤니티에서 지원됩니다.
====



이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능은 사용 가능합니다. 
* {cross} 이 기능은 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후 사용 가능 여부가 결정될 수 있습니다.
