= 지원되는 {raspberrypios} 기능
:description: Raspberry Pi OS 클라이언트에서 시스템 패키지, 등록, 원격 명령, 시스템 패키지 상태를 포함한 다양한 기능을 구성하고 사용하는 방법을 배웁니다.
:revdate: 2025-06-30
:page-revdate: {revdate}
:page-author: SUSE 제품 및 솔루션 문서 팀
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

아래 표는 {raspberrypios} 클라이언트에서 다양한 기능의 사용 가능 여부를 정리한 것입니다.

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
{productname}에서 {raspberrypios}에 대한 지원이 기술 미리보기로 제공됩니다.
====
endif::[]

[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {raspberrypios}은 {raspberrypios} 커뮤니티에서 지원됩니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능은 사용 가능합니다.
* {cross} 이 기능은 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후 사용 가능 여부가 결정될 수 있습니다.
