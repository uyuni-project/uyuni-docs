{webui} 대신에 명령줄(CLI)을 사용하여 클라이언트를 프록시에 등록할 수 있습니다.
이 절차를 수행하려면 등록 전에 클라이언트에 Salt 패키지가 설치되어 있어야 합니다.
SLE 12 기반 클라이언트의 경우 `Advanced Systems Management` 모듈도 활성화해야 합니다.



.절차 명령줄(CLI)을 사용하여 클라이언트를 프록시에 등록

. 다음 위치에 있는 클라이언트 구성 파일을 선택합니다.
+
----
/etc/salt/minion
----
+
또는:
+
----
/etc/salt/minion.d/NAME.conf
----
+
이 파일을 종종 minion 파일이라고도 합니다.
. 프록시 FQDN을 클라이언트 구성 파일에 `master`로 추가하십시오:
+
----
master: PROXY123.EXAMPLE.COM
----

. `salt-minion` 서비스를 다시 시작하십시오:
+
----
systemctl restart salt-minion
----
. 서버에서 새로운 클라이언트 키를 수락한 후, `<client>`를 클라이언트 이름으로 대체하십시오:
+
----
salt-key -a '<client>'
----
