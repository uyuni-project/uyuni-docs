[[oracle-registration-overview]]
= Oracle Client Registration
:description: Learn how to register and manage Oracle Linux clients via the method specific to each clients operating system with a SUSE Manager Server
:revdate: 2024-10-29
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 4.3

You can register {oracle} clients to your {productname} Server.
The method and details varies depending on the operating system of the client.

Before you start, ensure that the client has the date and time synchronized correctly with the {productname} Server.

You must also have created an activation key.
For more information about creating activation keys, see xref:client-configuration:activation-keys.adoc[].

For more information about migrating {oracle} to {sll}, see xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[].
