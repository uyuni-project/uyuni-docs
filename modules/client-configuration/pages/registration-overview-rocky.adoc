[[rocky-registration-overview]]
= {rocky} Client Registration
:description: Register your Rocky Linux clients to a SUSE Multi-Linux Manager Server with a supported operating system and activation key
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

You can register {rocky} clients to your {productname} Server.
The method and details vary depending on the operating system of the client.

Before you start, ensure that the client has the date and time synchronized correctly with the {productname} Server.

You must also have created an activation key.

* For more information about creating activation keys, see xref:client-configuration:activation-keys.adoc[].
* For more information about migrating {rocky} to {sll}, see xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[].
