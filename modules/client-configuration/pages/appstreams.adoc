= AppStream Modules
:description: The Server filters packages for Clients subscribed to an AppStream channel based on enabled AppStream module settings.
:revdate: 2024-05-03
:page-revdate: {revdate}

If you are managing a client that is subscribed to an AppStream channel, {productname} will filter packages available to the client depending on its currently enabled set of AppStream modules.
