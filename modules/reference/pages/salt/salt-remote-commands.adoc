= Remote Commands
:description: Learn how to issue commands to individual Salt clients or groups matching a specified search term via the Remote Commands feature
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 5.0

The menu:Salt[Remote Commands] section allows you to perform remote commands on one or more of your Salt clients.
Remote commands allows you to issue commands to individual Salt clients, or to all clients that match a search term.

For more information about remote commands, see xref:administration:actions.adoc[].
