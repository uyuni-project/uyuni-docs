= Audit Menu
:description: Manage security updates on client systems using the Audit Menu, which includes tools for finding and updating CVE patches, subscription matching, and OpenSCAP
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

The [guimenu]``Audit`` menu provides access to features for managing security updates on your clients.
Audit tasks include finding and updating clients with the latest CVE patches, subscription matching, and managing OpenSCAP scans.
