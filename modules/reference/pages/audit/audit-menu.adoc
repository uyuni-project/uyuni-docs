= Audit Menu
:description: Learn how to manage security updates on client systems effectively by using the Audit menu features to identify and apply the latest CVE patches
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 5.0

The [guimenu]``Audit`` menu provides access to features for managing security updates on your clients.
Audit tasks include finding and updating clients with the latest CVE patches, subscription matching, and managing OpenSCAP scans.
