= Reactivation
:description: Learn how to generate reactivation keys for clients in your SUSE Manager setup, regaining full access thereby
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 5.0

The menu:Systems[Details > Reactivation] section allows you to generate reactivation keys for the selected client.
Reactivation keys can be used once only to re-register the client and regain all {productname} settings.

For more information about reactivation keys, see xref:client-configuration:activation-keys.adoc[].
