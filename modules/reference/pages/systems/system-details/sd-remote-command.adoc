= Remote Command
:description: Run commands remotely on a client by configuring the system and accessing the Remote Command menu through its interface
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 5.0

The menu:Systems[Details > Remote Command] section allows you to run commands remotely on the selected client.

Before doing so, you must first configure the system to accept such commands.
For more information about using remote commands, see xref:administration:actions.adoc[].
