= Manage Repositories
:description: Configure and manage custom or third-party package repositories by linking them to an existing channel with YaST tools
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 4.3

The menu:Software[Manage > Repositories] section allows you to manage custom or third-party package repositories and link the repositories to an existing channel.

Click btn:[Create Repository] to create a new repository.

For more information about custom repositories and channels, see xref:administration:custom-channels.adoc[].
