= Archived Actions
:description: Access the Archived Actions section of the Schedule menu, where you can filter and browse through a list of archived actions
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

The menu:Schedule[Archived Actions] section shows actions that you have marked as archived.
Use the [guimenu]``Filter by Action`` field to search the list.
Completed or failed actions can be archived.

For more information about actions, see xref:administration:actions.adoc[].
