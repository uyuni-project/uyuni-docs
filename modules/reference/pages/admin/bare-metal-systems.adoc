= Bare Metal Systems
:description: Learn how to enable the bare metal feature in Admin Manager Configuration, then use it to provision clients for autoinstallation and explore the bare metal
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 4.3


In the menu:Admin[Manager Configuration > Bare-metal systems] section, you can turn on the bare metal feature.
This allows you to provision bare metal clients in preparation for autoinstallation.

For more information about bare metal provisioning, see xref:client-configuration:autoinst-intro.adoc[].
