= Show Tomcat Logs
:description: Learn how to configure and manage Apache Tomcat log files effectively using the Admin sections Show Tomcat Logs feature
:revdate: 2021-01-19
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 4.3

The menu:Admin[Show Tomcat Logs] section shows the Apache Tomcat log file.
You can also view the Tomcat log from the command prompt at [path]``/var/log/rhn/rhn_web_ui.log``.

[NOTE]
====
The menu:Admin[Show Tomcat Logs] section is only available if you are signed in to {productname} with an administrator account.
====
