= Admin Menu
:description: Learn how to manage your SUSE Manager configuration effectively using the Admin menus features for creating, organizing, managing users, and assigning tasks
:revdate: 2021-01-19
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 5.0

The [guimenu]``Admin`` menu provides access to features for managing {productname} configuration.
Configuration tasks include creating and managing organizations, users, and tasks.
You can also use the setup wizard to help configure {productname}.

[NOTE]
====
The [guimenu]``Admin`` menu is only available if you are logged in to {productname} with an administrator account.
====
