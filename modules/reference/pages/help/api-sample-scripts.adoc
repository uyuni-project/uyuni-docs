= API Sample Scripts
:description: Learn how to leverage example API calls in Ruby, Perl, and Python through sample scripts provided in the API Sample Scripts section of the Help menu
:revdate: 2025-05-28
:page-revdate: {revdate}

The menu:Help:[API > Sample Scripts] section contains example API calls for you to copy.
The scripts are written in Ruby, Perl, and Python.
