[[ref-spacecmd-toggle]]
= toggle
:description: Configure YaST tools to manage toggle functionality for various system settings via specific spacecmd functions that include display of confirmation messages
:revdate: 2023-06-21
:page-revdate: {revdate}

The following spacecmd functions are available to toggle functions on and off.

== toggle_confirmations

Toggle confirmation messages on/off.

[source]
----
usage: toggle_confirmations
----
