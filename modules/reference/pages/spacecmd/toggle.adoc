[[ref-spacecmd-toggle]]
= toggle
:description: Discover various toggle functions with YaST tools, which can be enabled and disabled using spacecmd commands
:revdate: 2023-06-21
:page-revdate: {revdate}

The following spacecmd functions are available to toggle functions on and off.



== toggle_confirmations

Toggle confirmation messages on/off.

[source]
----
usage: toggle_confirmations
----
