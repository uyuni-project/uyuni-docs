= history
:description: Navigate and manage previous commands efficiently with the history command in YaST tools, allowing for more efficient workflow
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

List recent commands using the `history` command.

[source]
--
spacecmd {SSM:0}> history
   1  help
   2  api
   3  exit
   4  help
   5  time --help
   6  quit
   7  clear
spacecmd {SSM:0}>
--
