[[ref-spacecmd-history]]
= history
:description: Explore how to navigate and utilize recent system commands efficiently within spacecmd sessions using the history command
:revdate: 2020-09-11
:page-revdate: {revdate}

List recent commands using the `history` command.

[source]
--
spacecmd {SSM:0}> history
   1  help
   2  api
   3  exit
   4  help
   5  time --help
   6  quit
   7  clear
spacecmd {SSM:0}>
--
