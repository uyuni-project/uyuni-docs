[[ref-spacecmd-whoamitalkingto]]
= whoamitalkingto
:description: Learn about retrieving and understanding the server hostname associated with your current Spacecmd connection using the built-in WhoAmITalkingTo function within
:revdate: 2023-06-21
:page-revdate: {revdate}

The following spacecmd function is available for returning the server hostname.



== whoamitalkingto

Return the server hostname that spacecmd is connected with.

[source]
----
spacecmd {SSM:0}> whoamitalkingto
MGR_SERVER_HOSTNAME
----
