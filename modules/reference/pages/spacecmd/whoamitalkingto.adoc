= whoamitalkingto
:description: Learn how to return the Server hostname that spacecmd is connected with using the whoamitalkingto function.
:revdate: 2023-06-21
:page-revdate: {revdate}

The following spacecmd function is available for returning the server hostname.



== whoamitalkingto


Return the server hostname that spacecmd is connected with.

[source]
----
spacecmd {SSM:0}> whoamitalkingto
MGR_SERVER_HOSTNAME
----
