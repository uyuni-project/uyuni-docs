[[ref-spacecmd-whoamitalkingto]]
= whoamitalkingto
:description: Use the specific function command to retrieve and understand the server hostname associated with your current spacecmd connection
:revdate: 2023-06-21
:page-revdate: {revdate}

The following spacecmd function is available for returning the server hostname.



== whoamitalkingto

Return the server hostname that spacecmd is connected with.

[source]
----
spacecmd {SSM:0}> whoamitalkingto
MGR_SERVER_HOSTNAME
----
