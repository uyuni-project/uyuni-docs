= whoami
:description: Learn how to use the whoami command to retrieve your current spacecmd username with step-by-step guidance and example usage in spacecmd
:revdate: 2023-06-21
:page-revdate: {revdate}

The following command is available for returning the currently logged spacecmd username.



== whoami

Print the currently logged spacecmd user.

[source]
----
spacecmd {SSM:0}> whoami
admin
----
