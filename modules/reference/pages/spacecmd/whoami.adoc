[[ref-spacecmd-whoami]]
= whoami
:description: Use the whoami command to retrieve and display your current logged-in username on the Spacecmd system
:revdate: 2023-06-21
:page-revdate: {revdate}

The following command is available for returning the currently logged spacecmd username.



== whoami


Print the currently logged spacecmd user.

[source]
----
spacecmd {SSM:0}> whoami
admin
----
