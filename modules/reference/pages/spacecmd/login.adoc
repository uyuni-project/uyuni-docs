= login
:description: Configure your connection to the SUSE manager server by using the spacecmd command with the --login option, specifying the required username
:revdate: 2020-09-11
:page-revdate: {revdate}

Connect as a specific user to the SUSE manager server.

[source]
--
# spacecmd -- login <USERNAME>
--
