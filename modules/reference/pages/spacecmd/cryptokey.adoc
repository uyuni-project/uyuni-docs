[[ref-spacecmd-cryptokey]]
= cryptokey
:description: Use the spacecmd commands to manage and manipulate various types of cryptographic keys, including creating, deleting, displaying details, and listing all
:revdate: 2021-08-10
:page-revdate: {revdate}

The following spacecmd commands are available for use with cryptographic keys.



== cryptokey_create

Create a cryptographic key.

[source]
--
usage: cryptokey_create [options]

options:
  -t GPG or SSL
  -d DESCRIPTION
  -f KEY_FILE
--



== cryptokey_delete

Delete a cryptographic key.

[source]
--
usage: cryptokey_delete NAME
--



== cryptokey_details

Show the contents of a cryptographic key.
[source]
--
usage: cryptokey_details KEY ...
--



== cryptokey_list

List all cryptographic keys (SSL, GPG).

[source]
--
usage: cryptokey_list
--
