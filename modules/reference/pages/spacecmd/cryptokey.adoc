= cryptokey
:description: Master the management and manipulation of cryptographic keys using these essential spacecmd commands create, delete, view, and list SSL and GPG keys securely
:revdate: 2021-08-10
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager

The following spacecmd commands are available for use with cryptographic keys.



== cryptokey_create


Create a cryptographic key.

[source]
--
usage: cryptokey_create [options]

options:
  -t GPG or SSL
  -d DESCRIPTION
  -f KEY_FILE
--



== cryptokey_delete

Delete a cryptographic key.

[source]
--
usage: cryptokey_delete NAME
--



== cryptokey_details

Show the contents of a cryptographic key.
[source]
--
usage: cryptokey_details KEY ...
--



== cryptokey_list

List all cryptographic keys (SSL, GPG).

[source]
--
usage: cryptokey_list
--
