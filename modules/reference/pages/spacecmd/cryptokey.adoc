[[ref-spacecmd-cryptokey]]
= cryptokey
:description: Learn about managing and maintaining your systems cryptographic keys using spacecmd commands, which offer options for creating, deleting, showing, listing,
:revdate: 2021-08-10
:page-revdate: {revdate}

The following spacecmd commands are available for use with cryptographic keys.



== cryptokey_create

Create a cryptographic key.

[source]
--
usage: cryptokey_create [options]

options:
  -t GPG or SSL
  -d DESCRIPTION
  -f KEY_FILE
--



== cryptokey_delete

Delete a cryptographic key.

[source]
--
usage: cryptokey_delete NAME
--



== cryptokey_details

Show the contents of a cryptographic key.
[source]
--
usage: cryptokey_details KEY ...
--



== cryptokey_list

List all cryptographic keys (SSL, GPG).

[source]
--
usage: cryptokey_list
--
