= Images Profiles
:description: Learn how to manage and configure image profiles, create new ones, update existing profiles, perform bulk actions, and delete unnecessary profiles
:revdate: 2020-09-11
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Manager
:page-product-version: 4.3


The menu:Images[Profiles] section displays your current image profiles.

Click btn:[Create] to create a new image profile.
Click btn:[Refresh] to update the list.

Perform bulk actions by checking profiles in the list.
Click btn:[Delete] to bulk delete profiles.

For more information about images, see xref:administration:image-management.adoc[].
