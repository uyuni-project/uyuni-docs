== 序文 


このガイドでは、{productname}サーバおよびプロキシをデプロイ、アップグレード、管理するための包括的な手順を段階的に説明します。

* **要件:**スムーズなセットアップを確実に行うために必要なハードウェア、ソフトウェア、およびネットワークの前提条件を概説します。
* **デプロイメントとインストール:**{productname}をコンテナとしてデプロイし、初期設定を完了する手順を案内します。
* **アップグレードと移行:**ダウンタイムを最小限に抑えながら、{productname}のアップグレードおよび移行のプロセスを詳述します。
* **基本的なサーバ管理:**基本的なサーバ操作をカバーし、{productname}を効率的に始める手助けをします。
