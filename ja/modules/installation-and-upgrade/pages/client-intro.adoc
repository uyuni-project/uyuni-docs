[[client-upgrade]]
= クライアントのアップグレード
:description: YaSTツールを使用してクライアントのアップグレードを管理する方法を学び、SUSE Multi-Linux Manager Web UIを使用して最適な機能のためにバージョン管理を構成します。
:revdate: 2024-11-19
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


クライアントは、基盤となるオペレーティングシステムのバージョン管理システムを使用しています。
{suse}オペレーティングシステムを使用しているクライアントの場合、{productname} {webui}内でアップグレードを実行できます。

クライアントのアップグレードの詳細については、xref:client-configuration:client-upgrades.adoc[]を参照してください。
