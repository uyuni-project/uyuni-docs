= を利用したライブパッチ{productname}
:description: ダウンタイムなしで重要なカーネル更新およびCVEパッチを適用する方法を、SUSE Multi-Linux Managerのライブパッチ機能を利用して学びます。この機能は、SLESの両バージョンでサポートされています。

:page-revdate: {revdate}

カーネルを更新するには、通常、システムの再起動が必要です。
共通脆弱性および曝露（CVE）パッチはできるだけ早く適用すべきですが、もしダウンタイムを許容できない場合は、ライブパッチを利用してこれらの重要な更新を反映させ、再起動の必要を省くことができます。

ライブパッチの設定手順は、SLES{nbsp}12とSLES{nbsp}15で若干異なります。
このセクションに、両方の手順が記載されています。
