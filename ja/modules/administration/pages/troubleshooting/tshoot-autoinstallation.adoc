= 自動インストールのトラブルシューティング
:description: クライアントツールのソフトウェアを設定し、すべての必要なパッケージが指定されたチャネルを通じて自動的に利用できるようにします。
:revdate: 2025-01-06
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

通信にプレーンな{salt} (salt-minion)実装を使用する場合は、必要なすべてのパッケージが設定済みクライアントチャンネルで適切に利用できることを各自で確認する必要があります。
この実装を{productname}と一緒に使用することは推奨されません。

デフォルトの{salt}バンドル (venv-salt-minion) 実装では、クライアントベースチャンネルと互換性のある子チャンネルとしてクライアントツールチャンネルのみが必要です。
必要なすべてのパッケージは{salt} バンドルの一部となります。

* 自動インストールプロファイルのベースチャンネルに関するクライアントツールソフトウェアチャンネルを組織およびユーザで使用できることを確認します。
* ツールチャンネルが子チャンネルとして{productname}で使用できることを確認します。
* プレーンな{salt} (salt-minion) 実装を使用する場合のみ、関連するチャンネルで、必要なパッケージと依存関係が使用可能であることを確認します。
