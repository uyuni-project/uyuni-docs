= プロキシの接続およびFQDNのトラブルシューティング
:description: プロキシを介して接続されたクライアントが正しい接続状態を表示できるように、クライアント設定で追加のFQDNをグレインとして追加して構成します。
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

プロキシを介して接続されたクライアントが{webui}に表示されることがありますが、プロキシを介して接続されていることは表示されません。
これは、完全修飾ドメイン名（FQDN）を使用して接続していない場合や、プロキシが{productname}に認識されていない場合に発生する可能性があります。

この動作を修正するには、プロキシのクライアント設定ファイルでgrainとして追加のFQDNを指定します。

----
grains:
  susemanager:
    custom_fqdns:
      - name.one
      - name.two
----
