[[troubleshooting-cloned-salt-clients]]
= クライアントが複製したSaltクライアントのトラブルシューティング
:description: Saltクライアントのクローンエラーを解決するには、各システムを一意に識別できるようにマシンIDを調整し、正常に登録できるようにしてください。
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

ハイパーバイザ複製ユーティリティを使用していて、複製したSaltクライアントを登録しようすると、次のエラーが発生します。

----
We're sorry, but the system could not be found.
----

これは、新しくクローンされたシステムが既存の登録済みシステムと同じマシンIDを持っているために発生します。
このエラーを修正し、クローンシステムを正常に登録するために、手動で調整してください。


詳細情報と手順については、xref:administration:troubleshooting/tshoot-registerclones.adoc[]を参照してください。
