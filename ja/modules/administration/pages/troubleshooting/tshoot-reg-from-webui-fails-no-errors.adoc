= {webui}からの登録が失敗し、エラーが表示されない場合のトラブルシューティング
:description: Web UIからの登録に関する問題を、/var/log/salt-ssh.logのローカルログに記録されたエラーメッセージの確認でトラブルシューティングし、解決する方法を学びましょう。
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{webui}からの初期登録では、すべてのSaltクライアントがSalt SSHを使用しています。

その性質上、Salt SSHクライアントはサーバにエラーを報告しません。

ただし、Salt SSHクライアントは、`/var/log/salt-ssh.log`にローカルでログを保存し、エラーを確認するために利用できます。
