.プロシージャ:コマンドプロンプトでのソフトウェアチャンネルの追加
. 「`spacewalk-common-channels`コマンド」を使用すると、適切なチャンネルを追加できます。
  {productname} コンテナホストのコマンドプロンプトで、ルートとして次のコマンドを実行します。
  正しいアーキテクチャを指定してください：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-common-channels \
-a <architecture> \
<base_channel_name> \
<child_channel_name_1> \
<child_channel_name_2> \
... <child_channel_name_n>
----
. xref:administration:custom-channels.adoc#_custom_channel_synchronization[自動同期]がオフになっている場合、次のコマンドを使用してコンテナホストのチャネルを同期してください：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-repo-sync -p <base_channel_label>-<architecture>
----

. 続行前に、同期が完了していることを確認してください。
