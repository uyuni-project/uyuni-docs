.プロシージャ:コマンドプロンプトでのソフトウェアチャンネルの追加

. 「`spacewalk-common-channels`コマンド」を使用すると、適切なチャンネルを追加できます。
  {productname} コンテナホストのコマンドプロンプトで、ルートとして次のコマンドを実行します:

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-common-channels \
<base_channel_label> \
<child_channel_label_1> \
<child_channel_label_2> \
... <child_channel_label_n>
----

+

使用可能なすべてのリポジトリを一覧にするには、次のコマンドを実行します。

+

[source,shell]
----
mgrctl exec -- spacewalk-common-channels -l
----

. xref:administration:custom-channels.adoc#_custom_channel_synchronization[自動同期] がオフになっている場合、チャンネルを同期します:

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-repo-sync -p <base_channel_label>
----

. 続行前に、同期が完了していることを確認してください。
