[WARNING]
====
RHEL 10は_x86-64-v3_ CPUアーキテクチャを必要とします。

**Red Hat Enterprise Linux 10**は、サポートされる最小のx86-64マイクロアーキテクチャのベースラインを_x86-64-v3_に引き上げました。

_x86-64-v1_または_x86-64-v2_ CPU（例：Haswell以前のIntelまたはExcavator以前のAMDプロセッサ）で動作しているシステムは、RHEL 10を実行できず、サポートされません。

これは、物理ホストと仮想マシンの両方に影響します。
ハイパーバイザー（例：VMware EVC、KVMなど）は、ゲストVMにAVX/AVX2 CPU機能を公開するように構成する必要があります。そうしないと、RHEL 10のインストールまたはブートが失敗します。

RHEL 10をデプロイまたはアップグレードする前に、ハードウェアが_x86-64-v3_の要件を満たしていることを確認してください。
