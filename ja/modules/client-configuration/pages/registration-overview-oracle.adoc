[[oracle-registration-overview]]
= Oracleクライアントの登録
:description: Oracle Linuxクライアントを、各オペレーティングシステムに適した登録方法で、お使いのSUSE Multi-Linux Manager Serverに登録してください。
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{oracle}クライアントをお使いの{productname}サーバに登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバと正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {oracle}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
