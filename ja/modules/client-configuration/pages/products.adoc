[[products]]
= ソフトウェア製品
:description: SUSEのサブスクリプションを通じて利用可能なソフトウェア製品の範囲を閲覧し、その後、SUSE Multi-Linux Manager Web UIで選択してください。
:revdate: 2024-05-10
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}では、ソフトウェアが製品として提供されます。
あなたの{suse}サブスクリプションにより、さまざまな製品にアクセスでき、それらを{productname} {webui}で閲覧し、menu:Admin[セットアップウィザード> 製品]に移動して選択できます。

製品には、任意の数のソフトウェアチャネルが含まれています。
`Show product's channels`アイコンをクリックすると、製品に含まれるチャネルを確認できます。
製品を追加し、正常に同期が完了すると、製品が提供するチャネルにアクセスでき、{productname}サーバーおよびクライアントで製品内のパッケージを使用できます。
