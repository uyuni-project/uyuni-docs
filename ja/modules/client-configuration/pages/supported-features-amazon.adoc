= サポートされている{amazon}機能
:description: この包括的な表では、Amazon Linuxクライアントがサポートする機能と機能性について説明しています
:revdate: 2025-06-30
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


この表には、{amazon}クライアントのさまざまな機能の使用可否がリストされています。

[IMPORTANT]
====
クライアントで実行しているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。{amazon}はAmazonでサポートされています。
====

この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用可能です
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
