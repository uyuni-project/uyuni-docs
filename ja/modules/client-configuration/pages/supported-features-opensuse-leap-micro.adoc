= {leapmicro} クライアント機能
:description: openSUSE Leap Micro クライアント環境を、オペレーティングシステムパッケージの登録やパッケージなどの幅広い機能を使用して構成および管理します。
:revdate: 2024-02-01
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


[IMPORTANT]
====
{leapmicro} は {suse} コミュニティによってサポートされています。
====


この表のアイコンの意味は次のとおりです。

* {check}: 機能は利用可能です
* {cross}: 機能は利用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
