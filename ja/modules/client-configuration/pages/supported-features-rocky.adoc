= サポートされている{rocky}機能
:description: 異なるRocky Linuxバージョンの利用可能な機能を、さまざまなクライアントにおける現在の利用可能性、サポート状況、及び可能性の観点から比較します。
:revdate: 2024-09-30
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム

:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


この表には、{rocky}クライアントのさまざまな機能の使用可否がリストされています。


[IMPORTANT]
====
クライアントで実行しているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。{rocky}は{rocky}コミュニティでサポートされています。
====


この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用できます
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
