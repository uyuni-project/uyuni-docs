= クライアント設定ガイド
:description: SUSE Multi-Linux Managerでクライアントの登録と設定を構成し、クライアントを効率的に管理します。
:revdate: 2025-05-21
:page-revdate: {revdate}

// HTML Publication date 
**発行日:** {docdate}

== 序文


{productname}をインストールした後、最初のステップはクライアントの登録です。これらのクライアントの管理は、ワークフローの重要な部分になります。

{productname}は{sle}およびその他のLinux配布パッケージをサポートし、さまざまなハードウェアオプション間での互換性を提供します。

サポートされているクライアントと機能の完全なリストについては、xref:client-configuration:supported-features.adoc[サポートされているクライアントと機能]を参照してください。

このガイドでは、手動と自動両方のクライアント登録と設定について説明します。
