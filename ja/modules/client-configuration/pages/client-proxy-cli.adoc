{webui}の代わりに、コマンドラインを使用してクライアントをプロキシに登録できます。
この手順では、登録前にクライアントにSaltパッケージがインストールされている必要があります。
SLE 12ベースのクライアントの場合、`Advanced Systems Management`モジュールも有効にする必要があります。



.プロシージャ:コマンドラインを使用してクライアントをプロキシに登録する

. 次の場所にあるクライアント設定ファイルを選択します。
+
----
/etc/salt/minion
----
+
または
+
----
/etc/salt/minion.d/NAME.conf
----
+
これはminionファイルと呼ばれることもあります。
. プロキシFQDNを`master`としてクライアント環境設定ファイルに追加します。
+
----
master: PROXY123.EXAMPLE.COM
----

. `salt-minion`サービスを再起動します：
+
----
systemctl restart salt-minion
----
. サーバで、新しいクライアントキーを受け入れます。`<client>`をクライアントの名前に置き換えます：
+
----
salt-key -a '<client>'
----
