= {oes}クライアントの登録
:description: あなたのSUSE Multi-Linux Manager Serverを使用してOpen Enterprise Serverクライアントを登録する方法を学び、クライアント登録方法を理解することから始めます
:revdate: 2026-05-05
:page-revdate: {revdate}
:page-author: SUSE製品＆ソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{oes}クライアントを{productname}サーバーに登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバと正しく同期しているか確認してください。

アクティベーションキーも作成しておく必要があります。

アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
