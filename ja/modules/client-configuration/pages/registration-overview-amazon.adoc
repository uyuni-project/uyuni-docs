[[amazon-registration-overview]]
= {amazon}クライアントの登録
:description: さまざまなオペレーティングシステムをサポートする方法を使用して、SUSE Multi-Linux Manager サーバーに Amazon Linux クライアントを登録します。
:revdate: 2021-07-22
:page-revdate: {revdate}
:page-author: SUSE 製品＆ソリューション ドキュメンテーション チーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{amazon}クライアントを{productname}サーバーに登録できます。
その方法および詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバと正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
