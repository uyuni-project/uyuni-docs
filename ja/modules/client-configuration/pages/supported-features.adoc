[[supported-features]]
= サポートされているクライアントと機能
:description: SUSE Linux Enterprise Server for SAP、AlmaLinux、CentOS、Debian、Oracle Linux、Red Hatを含む、幅広いLinuxオペレーティングシステムを構成および管理します。
:revdate: 2026-05-21
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}は、さまざまなクライアント技術と互換性があります。
{sle}または他のLinuxオペレーティングシステムを実行しているSaltクライアントを、さまざまなハードウェアオプションでインストールできます。

このセクションには、サポートされているクライアントシステムの概要が含まれています。各クライアントで利用可能な機能の詳細なリストについては、次のページを参照してください。


[[supported-features-clients]]
== サポートされているクライアントシステム


クライアントオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。
バージョンとSPレベルは、{productname}でサポートされるために、一般サポート（通常またはLTSS）の下である必要があります。
サポートされている製品バージョンの詳細については、 https://www.suse.com/lifecycle.を参照してください。


サポートされているクライアントオペレーティングシステムは、この表に示されています。表のアイコンの意味は次のとおりです。

* {check}このオペレーティングシステムを実行しているクライアントは{suse}でサポートされています。
* {cross}このオペレーティングシステムを実行しているクライアントは{suse}ではサポートされていません。
* {question}クライアントは検討中であり、後日サポートされるかどうかは未定です。

[IMPORTANT]
====
クライアントで実行されているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。
====

== 追加のツールパッケージ

`spacewalk-utils` パッケージおよび `spacewalk-utils-extras` パッケージは、追加のサービスや機能を提供できます。ユーザーは、廃止されたツールの使用を停止し、ドキュメントで推奨されている代替品に切り替えることをお勧めします。

ifeval::[{mlm-content} == true]
`spacewalk-utils` パッケージは {productname} で {suse} によって完全にサポートされており、次のツールが含まれています：
endif::[]

[[spacewalk-utils]]
[cols="1,1,1", options="header"]
.Spacewalkのユーティリティ
|===

| ツール名
| 説明
| 非推奨ですか？

| `spacewalk-common-channels`
| {scclongform}が提供していないチャネルを追加します。
|

| `spacewalk-hostname-rename`
| {productname}サーバのホスト名を変更します。
|

| `spacewalk-clone-by-date`
| 特定の日付でチャネルをクローンします。これは廃止されたツールであり、ユーザーはCLM APIに切り替えるべきです。
| {check}

| `spacewalk-sync-setup`
| ISSのマスターおよびスレーブ組織のマッピングを設定します。
| {check}

| `spacewalk-manage-channel-lifecycle`
| チャネルのライフサイクルを管理します。これは廃止されたツールであり、ユーザーはCLM APIに切り替えるべきです。
| {check}

|===

endif::[]
