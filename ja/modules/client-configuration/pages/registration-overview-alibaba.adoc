[[alibaba-registration-overview]]
= {alibabaclo}クライアントの登録
:description: さまざまな方法を使用して、SUSE Multi-Linux Manager ServerでAlibaba Cloud Linuxクライアントを登録します。これらの方法は、クライアントのオペレーティングシステムによって異なります。
:revdate: 2022-08-12
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{alibabaclo}クライアントを{productname}サーバーに登録できます。
その方法および詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバーと正しく同期しているか確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
