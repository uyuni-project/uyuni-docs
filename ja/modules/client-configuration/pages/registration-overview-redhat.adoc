[[redhat-registration-overview]]
= Red Hatクライアントの登録
:description: SUSE® Multi-Linux Manager Serverを介して、CDNまたはRHUIを使用してRed Hatクライアントを安全かつシームレスに登録します。
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{rhel}クライアントを{productname}サーバーに登録するには、{redhat}コンテンツ配信ネットワーク（CDN）または{redhat}更新インフラストラクチャ（RHUI）を使用できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {rhel}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
