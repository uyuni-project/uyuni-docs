[[ubuntu-registration-overview]]
= Ubuntuクライアントの登録
:description: Ubuntuクライアントを、オペレーティングシステムに適した方法でSUSE Multi-Linux Manager サーバに登録するように設定します。
:revdate: 2020-08-27
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}サーバに{ubuntu}クライアントを登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントで{productname}サーバと日時が正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
