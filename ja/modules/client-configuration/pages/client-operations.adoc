= クライアントの操作
:description: システムセットマネージャを使用して、個々のSUSE Multi-Linux Managerクライアントまたはクライアントのグループを管理し、「設定」で高度なクライアントオプションを設定します。
:revdate: 2024-03-05
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

登録、アップグレード、ソフトウェアのインストール、またはクライアントの削除に加えて、他の操作も実行できます。{productname} クライアントは個別にまたはクライアントのセットとして管理できます。
クライアントのセットは、システムセットマネージャを使用して一度限りの操作としてグループ化するか、システムグループを使用して恒久的にグループ化することができます。

高度なクライアント設定オプションのために、一般的な設定管理が利用可能です。
{productname} {webui}を使用して、カスタムシステム情報を取得し、クライアントの電源をオンまたはオフにし、再起動することができます。

以下のセクションでは、これらの各操作について詳細に説明します。
