= サポートされている{sll}機能
:description: SUSE Liberty Linuxクライアントでサポートされている機能の包括的なリストを表示します。システムパッケージ、登録を含みます。
:revdate: 2026-04-20
:page-revdate: {revdate}
:page-author: SUSE 製品およびソリューションドキュメンテーションチーム

:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

この表には、{sll}クライアントのさまざまな機能の使用可否がリストされています。

[NOTE]
====
{sll}クライアントは、時々{sleses}（SLES ES）、Liberty、RES、または{redhat} {es}とも呼ばれます。
====

[IMPORTANT]
====
クライアントで実行しているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。{sll}は{suse}でサポートされています。
====

この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用可能です
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
