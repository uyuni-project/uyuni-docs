[[centos-registration-overview]]
= CentOSクライアントの登録
:description: CentOSクライアントをSUSE Multi-Linux Manager サーバーに登録するには、クライアントの日時をサーバーと同期させ、アクティベーションキーを作成する必要があります。
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}サーバーに{centos}クライアントを登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {centos}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
