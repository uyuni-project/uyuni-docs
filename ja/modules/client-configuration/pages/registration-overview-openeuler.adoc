= {openeuler}クライアントの登録
:description: さまざまなオペレーティングシステムに適した登録方法を介して、openEulerクライアントをSUSE Multi-Linux Manager サーバーに構成します。

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
{productname}では、{openeuler}のサポートがテクニカルプレビューとして利用可能です。
====
endif::[]

{openeuler}クライアントを{productname}サーバーに登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバーと日時を正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
