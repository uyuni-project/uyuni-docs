= システムの種類
:description: クライアントシステムは、ベースシステムの種類を割り当て、仮想化やコンテナビルドホストの機能を追加することで構成します。
:revdate: 2023-11-23
:page-revdate: {revdate}
:page-author: SUSE 製品＆ソリューション ドキュメンテーション チーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

クライアントはシステムの種類によって分類されます。
すべてのクライアントには、ベースシステムの種類とアドオンシステムの種類の両方を割り当てることができます。

ベースシステムの種類は、すべてのクライアントで`Salt`です。

アドオンの種類には、仮想ホストとして動作するクライアント向けの`Virtualization Host`と、ビルドホストとして動作するクライアント向けの`Container Build Host`があります。

アドオンシステムの種類を調整するには、menu:システム[システムリスト> システムの種類]に移動します。
アドオンの種類を変更したいクライアントをチェックし、`Add-On System Type`を選択し、btn:[追加]またはbtn:[削除]のいずれかをクリックします。
