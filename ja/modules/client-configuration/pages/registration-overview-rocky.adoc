[[rocky-registration-overview]]
= {rocky}クライアントの登録
:description: サポートされているオペレーティングシステムとアクティベーションキーを使用して、Rocky LinuxクライアントをSUSE Multi-Linux Managerサーバーに登録してください。
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{rocky}クライアントを{productname}サーバーに登録できます。
その方法および詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントの日時が{productname}サーバーと正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。

* アクティベーションキーの作成に関する詳細については、xref:client-configuration:activation-keys.adoc[]を参照してください。
* {rocky}を{sll}に移行することに関する詳細については、xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]を参照してください。
