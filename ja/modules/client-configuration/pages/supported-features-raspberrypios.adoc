:description: Raspberry Pi OSクライアントのさまざまな機能を構成し、使用する方法を学びます。これには、システムパッケージ、登録、リモートコマンド、システムパッケージの状態が含まれます。

この表には、{raspberrypios}クライアントのさまざまな機能の使用可否がリストされています。

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
{productname}では、{raspberrypios}のサポートがテクニカルプレビューとして利用可能です。
====
endif::[]

[IMPORTANT]
====
クライアントで実行しているオペレーティングシステムは、オペレーティングシステムを提供している組織によってサポートされています。{raspberrypios}は{raspberrypios}コミュニティによってサポートされています。
====

この表のアイコンの意味は次のとおりです。

* {check}: 機能は使用できます
* {cross}: 機能は使用できません
* {question}: 機能は検討中であり、後日利用可能になるかどうかは未定です
