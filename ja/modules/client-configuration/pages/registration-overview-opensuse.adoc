[[opensuse-registration-overview]]
= openSUSEクライアントの登録
:description: openSUSEおよびopenSUSE Leap Microクライアントを正しいセットアップでSUSE Multi-Linux Manager Serverに登録します。
:revdate: 2024-02-22
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

{productname}サーバに{opensuse}および{leapmicro}クライアントを登録できます。
そのメソッドおよび詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントで{productname}サーバと日時が正しく同期していることを確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。


[IMPORTANT]
====
{productname}サーバを自分自身に登録しないでください。
{productname}サーバは個別に管理する必要があるか、別の{productname}サーバを使用して管理する必要があります。
複数のサーバを使用することに関する詳細は、xref:specialized-guides:large-deployments/hub-multi-server.adoc[]を参照してください。
====
