= Raspberry Pi OSクライアントの登録
:description: Raspberry Pi OSクライアントを、サーバとの時刻同期とオペレーティングシステムの利用を通じてSUSE Multi-Linux Managerサーバに登録する方法を学びます。
:revdate: 2025-06-30
:page-revdate: {revdate}
:page-author: SUSE製品およびソリューションドキュメンテーションチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
{productname}では、{raspberrypios}のサポートがテクニカルプレビューとして利用可能です。
====
endif::[]

{raspberrypios}クライアントを{productname}サーバに登録できます。
方法と詳細は、クライアントのオペレーティングシステムによって異なります。

始める前に、クライアントが{productname}サーバと正しく日時を同期しているか確認してください。

アクティベーションキーも作成しておく必要があります。
アクティベーションキーの作成に関する詳細は、xref:client-configuration:activation-keys.adoc[]を参照してください。
