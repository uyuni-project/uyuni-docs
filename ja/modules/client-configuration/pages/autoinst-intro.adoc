[[autoinstallation]]
= オペレーティングシステムのインストール
:description: SUSE Multi-Linux Managerを使って、インプレース、ネットワークブート、リムーバブルなどのさまざまな方法でオペレーティングシステムをインストールし、クライアントを管理する方法を学びます。
:revdate: 2025-05-20
:page-revdate: {revdate}
:page-author: SUSEプロダクト＆ソリューションドキュメントチーム
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

一般的に、すでに稼働しているクライアントを登録します。
これらのマシンは、{productname}に登録する直前に手動でインストールしたか、{productname}を環境に追加する前にインストールされた既存のシステムである可能性があります。

または、{productname}を使用して、オペレーティングシステムをインストールし、{productname}に一度に登録することもできます。
この方法は部分的または完全に自動化されているため、インストーラーの質問に答える時間を節約でき、特に多くのクライアントをインストールして登録したい場合に便利です。

{productname}からオペレーティングシステムをインストールするには、いくつかの方法があります：

* 登録済みのクライアントでインプレースインストールを行う
* PXEブートを使用してネットワークを通じてインストールする
* インストール用CD-ROMまたはUSBメモリを準備し、そのメディアでコンピュータをブートする
* {productname} {smr}ソリューションの一部として。

インプレース再インストール方法は、以前のオペレーティングシステムがクライアントにすでにインストールされており、クライアントがすでに{productname}に登録されていることを前提としています。

インプレースインストール方法に関する情報は、xref:client-configuration:autoinst-reinstall.adoc[登録済みシステムの再インストール]を参照してください。

ネットワークブートインストール方法は、未フォーマットのマシンで動作します。
ただし、特定のネットワーク構成でのみ実行できます：

* {productname}サーバーまたはそのプロキシのいずれかが、インストール対象のマシンと同じローカルネットワーク上にあるか、または、すべての中間ルーターを通過できるDHCPリレーがある場合。
* 新しいDHCPサーバをセットアップするか、既存のDHCPサーバを設定することができる。
* インストール対象のクライアントがPXEブートに対応しており、PXEブートを実行するように設定することができる。

//For information about the network boot method, see xref:client-configuration:autoinst-pxeboot.adoc[Install via the Network].

リムーバブルメディア方式を使用すると、これらのネットワーク制約を回避できます。
ただし、マシンがCD-ROMやUSBキーを読み込んで、それらから起動できることが前提です。
クライアントマシンへの物理的アクセスも必要です。

リムーバブルメディア方式に関する情報は、xref:client-configuration:autoinst-cdrom.adoc[CD-ROMまたはUSBキーを介してインストール]を参照してください。

[NOTE]
====
{ubuntu}および{debian}クライアントの自動インストールはサポートされていません。
これらのオペレーティングシステムは手動でインストールする必要があります。
====

{productname}の自動インストール機能は、{cobbler}というソフトウェアに基づいています。
{cobbler}の詳細については、 https://cobbler.readthedocs.io.を参照してください。

[NOTE]
====
{suse}は、{productname} {webui}内で利用可能な{cobbler}機能、または{productname} APIを通じてのみサポートされています。
Cobblerがサポートする唯一のコマンドラインのコマンドは`buildiso`です。
ここに記載されているのは、サポートされている機能のみです。
====
