The Systems  Details  Notes section lists notes about the selected client.

Click Create Note to create a new note. Give your note a subject, and add any details you require. Click Create to save the note.

Notes are listed in the table with a time stamp of when they were last edited. Click the subject of a note to edit or delete it.
