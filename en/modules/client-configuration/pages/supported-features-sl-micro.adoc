= Supported {sl-micro} Client Features
:description: Explore the supported features of SL Micro Client, including operating system packages, registration, and installation options
:revdate: 2024-07-18
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


[IMPORTANT]
====
The operating system you run on a client is supported by the organization that supplies the operating system.
{sl-micro} is supported by {suse}.
====

[IMPORTANT]
====
{sl-micro} is only supported as regular minion ([systemitem]``default`` contact method) for the time being. We are working on managing it as Salt SSH client ([systemitem]``salt-ssh`` contact method), too.
====

* {check} the feature is available
