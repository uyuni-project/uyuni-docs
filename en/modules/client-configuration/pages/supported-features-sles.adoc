= Supported {slsa} and {opensuse} Features
:description: Compare feature availability for SUSE and openSUSE clients across different operating system variants
:revdate: 2024-05-10
:page-revdate: {revdate}
:page-author: SUSE Product & Solution Documentation Team

:page-product-name: SUSE Multi-Linux Manager

This table lists the availability of various features on {suse} and {opensuse} clients.
This table covers all variants of the {sle} operating system, including {slsa}, {sleda}, {sles} for SAP, and {sles} for HPC.

[IMPORTANT]
====
The operating system you run on a client is supported by the organization that supplies the operating system.

* {sle} is supported by {suse}.
** {suse} only supports products that are under general support or Long-term Support (LTS).
* {opensuse} is supported by the {suse} community.

====

The icons in this table indicate:

* {check} the feature is available
* {cross} the feature is not available
* {question} the feature is under consideration, and may or may not be made available at a later date
