[NOTE]
====
{productname} server hosts that are hardened for security may restrict execution of files from the `/tmp` folder.
In such cases, as a workaround, export the `TMPDIR` environment variable to another existing path before running `mgradm`.

For example:

[source,shell]
----
export TMPDIR=/path/to/other/tmp
----

In {productname} updates, tools will be changed to make this workaround unnecessary.
====
