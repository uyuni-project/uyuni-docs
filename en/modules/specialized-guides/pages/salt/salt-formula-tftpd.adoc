[[tftpd-formula]]
= TFTPd Formula
:revdate: 2025-06-12
:page-revdate: {revdate}

The TFTPd formula is used to configure the TFTP service on the {productname} {smr} branch server.

[IMPORTANT]
====
The formula is used only for connecting {productname} Proxy 4.3. with {productname} Server {productnumber}.
Do not use this formula if you want to connect {productname} Proxy {productnumber}.
====

.Procedure: Configuring TFTP

. In the {susemgr} {webui}, open the details page for the branch server, and navigate to the `Formulas` tab.
. Select the `Tftpd` formula, and click btn:[Save].
. Navigate to the menu:Formulas[Tftpd] tab, and set these parameters:
* In the `Internal Network Address` field, enter the IP address of the branch server (for example: `192.168.1.5`).
* In the `TFTP Base Directory` field, enter the path to the {saltboot} directory (for example, `/srv/saltboot`).
* In the `Run TFTP Under User` field, enter `saltboot`.
. Click btn:[Save Formula] to save your configuration.
. Apply the highstate.
