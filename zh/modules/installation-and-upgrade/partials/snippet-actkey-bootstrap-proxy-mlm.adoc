== 为代理创建激活密钥


.过程：升级创建激活密钥
[role=procedure]
_____

. 导航到 menu:系统[激活密钥]，然后单击 btn:[创建密钥]。

. 为代理主机创建一个激活密钥，使用 {sl-micro} {microversion} 或 {sles} {bci-mlm} 作为父通道。
  为代理主机创建激活密钥并使用 \{micro} \{microversion} 作为父通道。此密钥应包括所有建议的通道以及用作扩展子通道的代理。

. 继续以 `default` 客户端身份启动代理主机。

_____

[WARNING]
====
确保代理仅分配给原始供应商通道。

目前不支持分配克隆通道。
====
