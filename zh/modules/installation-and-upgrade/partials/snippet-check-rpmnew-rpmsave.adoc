[NOTE]
====
在某些升级中，可能会生成多个 `*.rpmnew` 和 `*.rpmsave` 文件。这些文件的存在或数量并不表示需要进行手动操作。在 SUSE Multi-Linux Manager 容器环境中，所需的配置更改会在升级过程中自动应用。这些文件的创建是由于打包的默认设置与现有配置文件之间存在差异，也可能包含不打算合并的内部或信息性更改。如果配置更改需要手动操作，发行说明或升级文档中会明确记录。

请勿将这些文件视为升级后的检查清单，也不要将它们批量合并。仅在您积极排查特定问题并了解其包含的配置所产生的影响时，才查看文件。如果需要进行更改，应基于已知需求有意识地操作，而不是简单地复制这些文件中的差异。如果您不确定，请保持文件不变。
====
