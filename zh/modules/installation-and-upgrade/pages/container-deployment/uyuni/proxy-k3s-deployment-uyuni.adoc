[[installation-proxy-containers-k3s-k3s]]
== 安装 K3s


在容器主机上，安装 `K3s`（将 `<K3S_HOST_FQDN>` 替换为您的 K3s 主机的 FQDN）：

----
curl -sfL https://get.k3s.io | INSTALL_K3S_EXEC="--tls-san=<K3S_HOST_FQDN>" sh -
----


[[installation-proxy-containers-k3s-helm]]
== 安装工具

安装需要 `mgrpxy` 和 `helm` 软件包。

使用安装程序脚本安装 Helm：
----
curl -fsSL -o get_helm.sh https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3
chmod 700 get_helm.sh
./get_helm.sh
----

有关更多信息，请参见 https://helm.sh/docs/intro/install/#from-script.

`mgrpxy` 软件包可在容器 utils 储存库中获得。
选择与以下发行版匹配的一个： https://download.opensuse.org/repositories/systemsmanagement:/Uyuni:/Stable:/ContainerUtils/.

.过程
. 要进行安装，请运行以下命令：
+
----
transactional-update pkg install mgrpxy
----
. 重引导。



[[installation-proxy-containers-k3s-deploy]]
== 部署 {productname} 代理 Helm Chart

要配置 {productname} 代理 pod 使用的卷的存储，请为以下声明定义持久卷。
如果您不自定义存储配置，K3s 将自动为您创建存储卷。

永久性卷声明已命名为：

* `squid-cache-pv-claim`
* `/package-cache-pv-claim`
* `/tftp-boot-pv-claim`

按照  中的文档创建 {productname} 代理的配置xref:installation-and-upgrade:container-deployment/uyuni/proxy-container-setup-uyuni.adoc[]。
复制配置文件 `tar.gz`，然后安装：

----
mgrpxy install kubernetes /path/to/config.tar.gz
----

有关详细信息，请参见

* link:https://kubernetes.io/docs/concepts/storage/persistent-volumes/[]（Kubernetes）
* link:https://rancher.com/docs/k3s/latest/en/storage/[]（K3s）文档
