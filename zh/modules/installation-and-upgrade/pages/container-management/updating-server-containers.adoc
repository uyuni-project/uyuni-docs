.去除导入的证书
[NOTE]
====
在以前的版本中，数据库容器需要具有以下主题备用名称（SANs）的 SSL 证书：

* `db`
* `reportdb`

而现在的版本不再有这项要求。
数据库容器只需要外部完全限定的域名。
带有 `db` 和 `reportdb` SAN 的旧证书仍然可以使用。

同一证书可以用于主容器和数据库容器。

为了将新证书传递给升级命令，请使用 `--ssl-db-ca-root`、`--ssl-db-cert` 和 `--ssl-db-key` 参数。
====

.升级到特定版本
[NOTE]
====
如果您未指定标签参数，它将默认为升级到最新版本。
要升级到特定版本，请提供带有所需镜像标签的标签参数。
====

要查看 upgrade 命令及其参数的详细信息，请使用以下命令：
