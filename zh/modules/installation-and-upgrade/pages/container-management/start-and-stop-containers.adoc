= 启动和停止容器
:description: 使用简单的终端命令管理和重启您的 SUSE Multi-Linux Manager 5.1 服务器容器
:revdate: 2024-02-13
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

可使用以下命令重启、启动和停止 {productname} {productnumber} 服务器容器：


要 `restart`{productname}{productnumber} 服务器，请执行以下命令:

----
# mgradm restart
5:23PM INF Welcome to mgradm
5:23PM INF Executing command: restart
----

要 `start` 服务器，请执行以下命令:

----
# mgradm start
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: start
----

要 `stop` 服务器，请执行以下命令:

----
# mgradm stop
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: stop
----
