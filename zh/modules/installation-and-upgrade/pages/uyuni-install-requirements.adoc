== 代理要求

| 软件和硬件
| 细节
| 推荐

| {leapmicro} {microversion}

|

| RAM
| Test Server

|
| 生产服务器
| 至少 8{nbsp}GB

| `/`（根目录）
| 最低 40{nbsp}GB

|
| `/srv`
| 至少 100{nbsp}GB

|
| `/var/cache` (Squid)
| 至少 100{nbsp}GB
|===


{productname} 代理在 `/var/cache/` 目录中缓存软件包。
如果 `/var/cache/` 中可用空间不足，代理将去除旧的未使用软件包，并用更新的软件包替换它们。

鉴于这种行为：

* 代理上的 `/var/cache/` 目录越大，代理与 {productname} 服务器之间的流量就越少。

* 使代理上的 `/var/cache/` 目录与 `/var/spacewalk/` 服务器上的 {productname} 保持相同的大小，可以避免在首次同步后出现大量的流量。

* 与代理相比，`/var/cache/` 服务器上的 {productname} 目录可能较小。
    有关大小估算的指南，请参见 <<server-hardware-requirements>> 部分。
