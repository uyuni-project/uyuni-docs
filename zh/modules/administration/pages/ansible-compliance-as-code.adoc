== 使用 Ansible 剧本进行修复


需要一个 Ansible 控制节点。
有关更多信息，请参见 xref:administration:ansible-setup-control-node.adoc[]。

以下过程将介绍如何使用 Ansible 剧本运行修复任务。

.过程：升级使用 Ansible Playbook 进行修复。

. 在控制节点系统菜单中选择 menu:Ansible[Playbooks]，然后单击某个 Playbook，例如：

+

[source,shell]
----
sle15-playbook-stig.yml
----

. 要运行该 Playbook，请选择客户端的 `Inventory Path`，例如：

+

[source,shell]
----
/etc/ansible/sles15
----

+

单击 btn:[日程安排]。

. 在 `Events` 选项卡下检查已安排事件的状态。



如果 Playbook 位于不同的目录，您可以单击 `Setup Ansible Control Node` 链接了解如何添加它。
