= 镜像源软件包
:description: 通过在 rhn.conf 配置文件中全局启用此功能，配置 SUSE Multi-Linux Manager Server 以镜像源软件包，然后重启该服务器。
:revdate: 2024-05-10
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

如果您在本地构建自己的软件包，或者因法律原因需要软件包的源代码，可以在 {productname} Server 上镜像源软件包。

[NOTE]
====
镜像源软件包可能会消耗大量磁盘空间。
====

[IMPORTANT]
====
在服务器容器内运行步骤之前，请使用 `mgrctl term`。
====

.过程：升级镜像源软件包
. 打开 `/etc/rhn/rhn.conf` 配置文件，并添加此行：
+

----
server.sync_source_packages = 1
----
. 重启 Spacewalk 服务以应用更改：
+

----
mgradm restart
----

目前，此功能只能为所有储存库全局启用。
无法选择单个储存库进行镜像。

当此功能被激活后，源软件包将在下次储存库同步后在 {productname} {webui} 中可用。
它们作为二进制软件包的源显示，并可以直接从 {webui} 下载。
源软件包无法在使用 {webui} 的客户端上安装。
