= 使用 {productname} 进行实时修补
:description: 了解如何使用 SUSE Multi-Linux Manager 的实时修补功能，在无需停机的情况下应用关键内核更新和 CVE 补丁，该功能支持 SLES。
:revdate: 2024-12-02
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


执行内核更新通常需要重启系统。
应尽快应用 CVE 补丁，但如果您无法承受停机时间，则可使用实时修补在注入这些关键更新的同时免去重启。

设置实时修补的过程在 SLES{nbsp}12 和 SLES{nbsp}15 中略有不同。
这两个过程均在本节中有所记录。
