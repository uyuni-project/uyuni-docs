= 对克隆的 Salt 客户端进行查错
:description: 通过调整机器 ID 来解决 Salt 客户端中的克隆错误，从而确保每个系统都能被唯一标识，实现成功注册
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

如果您曾经使用过超级管理程序克隆实用程序，并尝试注册克隆的 Salt 客户端，您可能会收到以下错误：

----
We're sorry, but the system could not be found.
----

这是由于新克隆的系统与现有注册系统具有相同的机器 ID 所导致的。
您可以手动调整此设置以纠正错误并成功注册克隆系统。


有关更多信息和说明，请参见 xref:administration:troubleshooting/tshoot-registerclones.adoc[]。
