= 使用 {webui} 进行的注册故障排查失败，且未显示任何错误。
:description: 了解如何通过 Web UI 排查并解决注册问题，可通过检查 /var/log/salt-ssh.log 中的本地日志来查找错误消息。
:revdate: 2022-10-18
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

在 {webui} 进行初始注册时，所有 Salt 客户端均使用 Salt SSH。

由其性质决定，Salt SSH 客户端不会向服务器回报错误。

不过，Salt SSH 客户端会将日志保存在本地的 `/var/log/salt-ssh.log`，可以用于检查错误。
