有时，{webui} 在迁移后无法加载。
这通常是由于浏览器缓存造成的，如果新系统与旧系统具有相同的主机名和 IP 地址。
这种重复可能会混淆某些浏览器。

通过清除缓存并重新加载页面可以解决此问题。
在大多数浏览器中，您可以通过按 kbd:[Ctrl+F5] 快速完成此操作。
