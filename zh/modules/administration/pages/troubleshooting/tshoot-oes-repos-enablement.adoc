= 对启用 OES 储存库时出现的问题进行查错
:description: 学习如何使用 SUSE Multi-Linux Manager Server 程序来排除故障并启用 OES 储存库
:revdate: 2022-11-03
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

要在 {productname} Server 上启用 Open Enterprise Server (OES)，请按照所述的过程进行操作。
 
.过程：升级启用 OES 储存库
[role=procedure]

. 确保您拥有 Microfocus 提供的有权访问 OES 的镜像身份凭证。
+
. 登录至 {productname} Server。
+
. 其网址为 `Admin > Setup Wizard > Organization Credentials`。
+
. 确保您已拥有 {productname} 的 SUSE 身份凭证。
+
. 选择用于添加新身份凭证的选项，然后键入 Microfocus 身份凭证。
+
. 前往 `Admin > Setup Wizard > Organization Credentials` 并等待刷新操作完成。
+
. 在刷新后的产品列表中应该可以看到 OES。
  您现在可以像启用其他产品一样启用它。

有关 OES 的详细信息，请参见 https://www.microfocus.com/documentation/open-enterprise-server/.。
