.过程：升级在命令提示符下添加软件通道

. 使用 `spacewalk-common-channels` 命令您可以添加相应的频道。
  在 {productname} 容器主机的命令提示符下，以 root 身份执行以下命令：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-common-channels \
<base_channel_label> \
<child_channel_label_1> \
<child_channel_label_2> \
... <child_channel_label_n>
----

+

要列出所有可用储存库，请执行以下命令：

+

[source,shell]
----
mgrctl exec -- spacewalk-common-channels -l
----

. 如果 xref:administration:custom-channels.adoc#_custom_channel_synchronization[自动同步] 被关闭，请同步通道：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-repo-sync -p <base_channel_label>
----

. 确保同步已完成，然后再继续操作。
