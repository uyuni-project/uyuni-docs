.过程：升级在命令提示符下添加软件通道
. 使用 `spacewalk-common-channels` 命令，您可以添加相应的频道。
  在 {productname} 容器主机的命令提示符下，以 root 身份执行以下命令。
  确保指定正确的架构：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-common-channels \
-a <architecture> \
<base_channel_name> \
<child_channel_name_1> \
<child_channel_name_2> \
... <child_channel_name_n>
----
. 如果 xref:administration:custom-channels.adoc#_custom_channel_synchronization[自动同步] 被关闭，请使用以下命令在容器主机上同步这些通道：

+

[source,shell]
----
mgrctl exec -ti -- spacewalk-repo-sync -p <base_channel_label>-<architecture>
----

. 确保同步已完成，然后再继续操作。
