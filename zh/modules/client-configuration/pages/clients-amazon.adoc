目前支持的体系结构有：`x86_64` 和 `aarch64`。
有关支持的产品和体系结构的完整列表，请参见 xref:client-configuration:supported-features.adoc[]。

== 创建激活密钥

您需要创建与您的 {amazon} 通道关联的激活密钥。

有关激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。



== 管理 GPG 密钥

客户端使用 GPG 密钥在安装软件包之前检查其真实性。
只有受信任的软件包才能在客户端上安装。

[IMPORTANT]
====
信任 GPG 密钥对客户端的安全性至关重要。
决定需要哪些密钥并可以信任哪些密钥是管理员的任务。
由于在 GPG 密钥未被信任时无法使用软件通道，因此是否将通道分配给客户端取决于是否信任该密钥。
====

有关 GPG 密钥的更多信息，请参见 xref:client-configuration:gpg-keys.adoc[]。
