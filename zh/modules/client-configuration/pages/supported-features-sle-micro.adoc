= 支持的 {sle-micro} 功能
:description: 请通过我们的文档访问 SLE micro 操作系统所支持功能的完整列表
:revdate: 2024-07-18
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{sle-micro} 由 {suse} 支持。
====

[IMPORTANT]
====
目前仅支持将 {sle-micro} 作为常规 minion（`default` 联系方式）使用。我们也在努力将其作为 Salt SSH 客户端（`salt-ssh` 联系方式）进行管理。
====

此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
