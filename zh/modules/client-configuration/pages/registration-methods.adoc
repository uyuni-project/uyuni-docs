[[registration-methods]]
= 注册方法
:description: 使用各种方法（如 YaST 工具和命令行命令）将客户端注册到您的 SUSE 服务器
:revdate: 2024-02-28
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

有多种方法可将客户端注册到您的 {productname} 服务器。

* 对于少数客户端，我们建议使用 {productname} {webui} 对其进行注册。
    有关更多信息，请访问 xref:client-configuration:registration-webui.adoc[]。
* 如果您想更好地控制注册过程或必须注册许多客户端，我们建议您创建引导脚本。
    有关更多信息，请参见 xref:client-configuration:registration-bootstrap.adoc[]。
* 如果您想进一步提高注册过程的控制度，在命令行上执行单个命令可能更好。
    有关更多信息，请参见 xref:client-configuration:registration-cli.adoc[]。

注册前，客户端的日期和时间必须已与 {productname} 服务器正确同步。

您必须先创建一个激活密钥，以使用启动脚本或命令行方法。
有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。


[IMPORTANT]
====
请勿将 {productname} 服务器注册到自身。
{productname} 服务器必须单独管理，或使用另一个单独的 {productname} 服务器进行管理。
有关使用多个服务器的更多信息，请参见 xref:specialized-guides:large-deployments/hub-multi-server.adoc[]。
====
