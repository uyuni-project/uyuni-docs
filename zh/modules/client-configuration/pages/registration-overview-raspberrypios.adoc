= Raspberry Pi OS 客户端注册
:description: 学习如何通过与服务器同步时间并使用操作系统，将 Raspberry Pi OS 客户端注册到 SUSE Multi-Linux Manager 服务器。
:revdate: 2025-06-30
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
在 {productname} 中，{raspberrypios} 的支持作为技术预览可用。
====
endif::[]

您可以将 {raspberrypios} 客户端注册到您的 {productname} 服务器。
方法和细节取决于客户端的操作系统。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。
有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。
