= {oes} 客户端注册
:description: 了解如何使用您的 SUSE Multi-Linux Manager 服务器注册 Open Enterprise Server 客户端，首先了解客户端注册方法
:revdate: 2026-05-05
:page-revdate: {revdate}

您可以将 {oes} 客户端注册到您的 {productname} 服务器。
方法和细节视客户端的操作系统而异。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。

有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。
