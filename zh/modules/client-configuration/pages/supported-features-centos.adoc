= 支持的 CentOS 功能
:description: 了解 CentOS 操作系统上支持的功能，包括系统软件包、远程命令等。
:revdate: 2024-09-30
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


下表列出了 {CentOS} 客户端上各种功能的可用性。


[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{centos} 由 {centos} 社区支持。
====



此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
