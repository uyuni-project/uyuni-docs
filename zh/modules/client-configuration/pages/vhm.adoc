[[virt-vhm]]
= 虚拟主机管理器
:description: 使用虚拟主机管理器配置虚拟化客户端，以收集在SUSE Multi-Linux Manager Web UI中显示的数据
:revdate: 2025-04-15
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

虚拟主机管理器 (VHM) 用于收集各种客户端类型的信息。

虚拟主机管理器可以用于收集私有或公有云实例，并将其组织成虚拟化组。
通过这种方式组织的虚拟化客户端，Taskomatic会收集客户端的数据以在{productname} {webui}中显示。
虚拟主机管理器还允许您在虚拟化客户端上使用订阅匹配。

您可以在您的{productname}服务器上创建一个虚拟主机管理器，并用它来清点可用的公有云实例。
您还可以使用虚拟主机管理器来管理使用{kube}创建的集群。

* 有关如何将虚拟主机管理器与亚马逊网络服务一起使用的更多信息，请参见xref:client-configuration:vhm-aws.adoc[]。
* 有关如何将虚拟主机管理器与微软Azure一起使用的更多信息，请参见xref:client-configuration:vhm-azure.adoc[]。
* 有关如何将虚拟主机管理器与Nutanix一起使用的更多信息，请参见xref:client-configuration:vhm-nutanix.adoc[]。
* 有关如何将虚拟主机管理器与VMWare vSphere一起使用的更多信息，请参见xref:client-configuration:vhm-vmware.adoc[]。
* 有关如何将虚拟主机管理器与其他主机一起使用的更多信息，请参见xref:client-configuration:vhm-file.adoc[]。
