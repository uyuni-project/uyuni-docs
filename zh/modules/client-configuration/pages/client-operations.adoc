= 客户端操作
:description: 使用系统集管理器管理单个 SUSE Multi-Linux Manager 客户端或客户端组，使用配置进行高级客户端选项的设置。

除了注册、升级、安装软件或删除客户端外，还可以执行其他操作。{productname} 客户端可以单独管理或作为一组客户端管理。
可以使用系统集管理器将一组客户端分组，以便进行一次性操作，或通过系统组进行永久分组。

对于高级客户端配置选项，可使用通用的配置管理功能。
您可以使用 {productname} {webui} 获取自定义系统信息，开启或关闭电源，并重启客户端。
