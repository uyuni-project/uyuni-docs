= 注册 {leapmicro} 客户端
:description: 了解如何注册运行 openSUSE Leap Micro 的客户端，并为可靠的边缘计算环境配置通道管理
:revdate: 2026-06-01
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

本节包含有关在 {x86_64} 和 {aarch64} 架构上注册运行 {leapmicro} 操作系统的客户端的信息。

{leapmicro} 是一个超可靠、轻量级的操作系统，专为边缘计算而构建。
它利用 SUSE Linux Enterprise 的企业级安全和合规组件，并将其与现代、不可变、开发者友好的操作系统平台相结合。

{leapmicro} 使用事务更新。
事务更新具有原子性（仅当所有更新都成功时，才会应用所有更新），且支持回滚。
它们不会影响正在运行的系统，因为在系统重启之前不会激活任何更改。
此信息显示在 menu:系统[详细信息> 概述] 子选项卡中。

有关事务更新和重启的更多信息，请参见 https://documentation.suse.com/sles/15-SP7/html/SLES-all/cha-transactional-updates.html.

== 注册客户端

[WARNING]
====
{leapmicro} 客户端在注册后需要重启。

在启动过程完成后，{leapmicro} 上禁用自动引导。此更改是由于间歇性自动重启而实施的，这可能发生在 Salt 能够传递应用启动 salt 状态的结果之前。

重启将在注册完成后自动安排，但它遵循默认的重启管理器维护窗口。
该窗口可能在客户端注册后几个小时。
为了加快 {leapmicro} 注册，请在注册脚本完成后手动重启客户端。
====
