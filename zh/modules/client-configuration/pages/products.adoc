[[products]]
= 软件产品
:description: 浏览通过您的SUSE订阅提供的软件产品范围，然后通过SUSE Multi-Linux Manager Web UI选择它们。
:revdate: 2024-05-10
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

在{productname}中，软件以产品的形式提供。
您的{suse}订阅允许您访问一系列不同的产品，您可以在{productname} {webui}中浏览和选择它们，通过导航到menu:Admin[设置向导> 产品]。

产品可以包含任意数量的软件频道。
单击`Show product's channels`图标以查看产品中包含的软件频道。
当您添加了一个产品并成功同步后，您可以访问该产品提供的软件频道，并可以在您的{productname}服务器和客户端上使用该产品中的软件包。
