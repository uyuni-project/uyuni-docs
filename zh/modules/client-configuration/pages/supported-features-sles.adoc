= 支持的 {slsa} 和 {opensuse} 功能
:description: 比较不同操作系统变体中 SUSE 和 openSUSE 客户端的功能可用性
:revdate: 2024-05-10
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1


下表列出了 {suse} 和 {opensuse} 客户端上各种功能的可用性。
该表涵盖了 {sle} 操作系统的所有变体，包括 {slsa}、{sleda}、{sles} 用于 SAP，以及 {sles} 用于 HPC。

[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。

* {sle} 由 {suse} 支持。
** {suse} 仅支持处于一般支持或长期支持 (LTS) 的产品。
* {opensuse} 由 {suse} 社区支持。

====

此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
