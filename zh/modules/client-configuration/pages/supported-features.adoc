[[supported-features-clients]]
== 支持的客户端系统


客户端操作系统由提供操作系统的组织支持。
版本和SP级别必须在一般支持（正常或LTSS）下，才能与{productname}一起获得支持。
有关支持的产品版本的详细信息，请参见 https://www.suse.com/lifecycle.


支持的客户端操作系统列在此表中。表中的图标表示：

* 运行此操作系统的{check}客户端受{suse}支持
* 运行此操作系统的{cross}客户端不受{suse}支持
* {question}客户端正在考虑之中，日后可能受支持，也可能不受支持。

[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。
====

== 其他工具软件包

`spacewalk-utils` 和 `spacewalk-utils-extras` 软件包可以提供额外的服务和功能。建议用户停止使用已弃用的工具，并切换到文档中推荐的替代工具。

ifeval::[{mlm-content} == true]
`spacewalk-utils` 软件包在 {productname} 中由 {suse} 完全支持，并包含以下工具：
endif::[]

[cols="1,1,1", options="header"]

|===

| 说明
| 已弃用？

| 添加 {scclongform} 未提供的频道

| `spacewalk-hostname-rename`
| 更改 {productname} 服务器的主机名
|

| `spacewalk-clone-by-date`
| 按特定日期克隆频道。这是一个已弃用的工具，用户应切换到 CLM API。
| {check}

| 设置 ISS 主从组织映射。
| {check}

| `spacewalk-manage-channel-lifecycle`
| 管理频道生命周期。这是一个已弃用的工具，用户应切换到 CLM API。
| {check}
