[[opensuse-registration-overview]]
= openSUSE 客户端注册
:description: 通过正确的设置，将 openSUSE 和 openSUSE Leap Micro 客户端注册到 SUSE Multi-Linux Manager 服务器。
:revdate: 2024-02-22
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

您可以将 {opensuse} 和 {leapmicro} 客户端注册到您的 {productname} 服务器。
方法和细节视客户端的操作系统而异。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。
有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。


[IMPORTANT]
====
请勿将 {productname} 服务器注册到自身。
{productname} 服务器必须单独管理，或使用另一个单独的 {productname} 服务器进行管理。
有关使用多个服务器的更多信息，请参见 xref:specialized-guides:large-deployments/hub-multi-server.adoc[]。
====
