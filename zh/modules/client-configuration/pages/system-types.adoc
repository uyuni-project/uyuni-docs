= 系统类型
:description: 通过分配基础系统类型并添加虚拟化或容器构建主机功能来配置您的客户端系统
:revdate: 2023-11-23
:page-revdate: {revdate}
:page-author: SUSE产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

客户端按系统类型进行分类。
每个客户端可以同时分配基础系统类型和附加系统类型。

所有客户端的基础系统类型为 `Salt`。

附加产品系统类型包括 `Virtualization Host`（适用于作为虚拟主机运行的客户端）和 `Container Build Host`（适用于作为构建主机运行的客户端）。

您可以通过导航到 menu:系统[系统列表> 系统类型] 来调整附加系统类型。
检查您想要更改附加产品系统类型的客户端，选择 `Add-On System Type`，然后点击 btn:[添加系统类型] 或 btn:[去除系统类型]。
