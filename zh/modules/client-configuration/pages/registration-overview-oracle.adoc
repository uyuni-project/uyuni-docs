= Oracle 客户端注册
:description: 使用适合其操作系统的正确注册方法，将您的 Oracle Linux 客户端注册到您的 SUSE Multi-Linux Manager Server。
:revdate: 2024-01-31
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1

您可以将 {oracle} 客户端注册到您的 {productname} 服务器。
方法和细节视客户端的操作系统而异。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。

* 有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。
* 有关将 {oracle} 迁移到 {sll} 的更多信息，请参见 xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]。
