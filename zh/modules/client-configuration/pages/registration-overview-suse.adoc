== 前期步骤或检查


开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。
有关创建激活密钥的更多信息，请参见 xref:client-configuration:activation-keys.adoc[]。



[IMPORTANT]
====
请勿将 {productname} 服务器注册到自身。
{productname} 服务器必须单独管理，或使用另一个单独的 {productname} 服务器进行管理。
有关使用多个服务器的更多信息，请参见 xref:specialized-guides:large-deployments/hub-multi-server.adoc[]。
====



[[suse-registration-overview-check]]
== 检查之前的注册状态

如果客户端之前已注册到 SCC、SMT 或 RMT，请在将客户端迁移到 `SUSEConnect` 服务器之前，使用 {productname} 去除该注册。
请不要忘记先去除旧的注册。
否则，在客户端迁移到更新版本的 {suse} 操作系统并使用 {productname} 后，不再需要的储存库将被添加到客户端。

要取消注册 {sle} 12 或 15 客户端，请在客户端上以 root 身份执行：

----
SUSEConnect --de-register
SUSEConnect --cleanup
----

有关详细清理命令和验证步骤的更多信息，请参见 https://www.suse.com/de-de/support/kb/doc/?id=000019054.
