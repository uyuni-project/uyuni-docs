[WARNING]
====
在 AWS 创建时，{openeuler} 实例在 `machine-id` 的 `/etc/machine-id` id 始终相同。
确保在实例创建后重新生成 `machine-id`。
有关详细信息，请参见 xref:administration:troubleshooting/tshoot-registerclones.adoc[]。
====



== 添加软件通道


在将 {openeuler} 客户端注册到您的 {productname} 服务器之前，您需要添加所需的软件通道，并同步这些通道。

目前支持的体系结构有：`x86_64` 和 `aarch64`。
