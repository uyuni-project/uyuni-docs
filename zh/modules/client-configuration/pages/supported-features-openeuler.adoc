= 支持的 {openeuler} 功能
:description: 这是更正后的句子，了解在 openEuler 客户端上支持的功能，包括系统软件包、注册选项、软件包安装
:revdate: 2026-06-09
:page-revdate: {revdate}
:page-author: SUSE 产品与解决方案文档团队
:page-image: https://www.suse.com/assets/img/suse-white-logo-green.svg
:page-product-name: SUSE Multi-Linux Manager
:page-product-version: 5.1
ifeval::[{mlm-content} == true]

:noindex:
endif::[]

下表列出了 {openeuler} 客户端上各种功能的可用性。

ifeval::[{mlm-content} == true]

[IMPORTANT]
====
在 {productname} 中，{openeuler} 的支持作为技术预览可用。
====

endif::[]



[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{openeuler} 由 {openeuler} 社区支持。
====



此表中图标的含意如下：

* {check} 该功能可用 
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
