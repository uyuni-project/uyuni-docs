== O que é {productname}?


// [#salt.gloss] may be used to create a tooltip for a glossary term: see branding/supplemental-ui/mlm/mlmcom/partials/footer-scripts.hbs

{productname} é uma solução para organizações que exigem controle robusto sobre manutenção e implantação de pacotes em seus servidores. Ele permite gerenciar grandes conjuntos de sistemas Linux e mantê-los atualizados, com gerenciamento automatizado de software, gerenciamento de ativos e provisionamento de sistema. Ele também permite manter um alto nível de segurança enquanto gerencia com eficácia os requisitos do ciclo de vida do sistema.

{productname} usa {salt} para fornecer configuração orientada a eventos e controle de gerenciamento. O Salt-master orquestra dezenas de milhares de clientes {salt} (clientes {productname}) usando execução remota.

{productname} oferece gerenciamento contínuo de sistemas clientes {sle}, {opensuse}, {rhel}, {centos}, {oracle}, {ubuntu}, {debian}, {amazon}, {almalinux} e {raspberrypios}. Eles podem estar no local ou em nuvem pública, nuvem privada, nuvem híbrida ou até mesmo em ambientes multinuvem.
