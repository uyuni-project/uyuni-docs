[[retail-upgrade-branchserver]]
= Upgrade {smr} Branch Server

This section describes upgrading a {smr} Branch Server to the next SP
(service pack).

{smr} Branch Server is a client system such as a {productname} Proxy with
additional {smr} features.

[NOTE]
====
Upgrade the {productname} Server before starting the {smr} upgrade.
====



.Procedure: Upgrading the {smr} Branch Server
. For general information about upgrading a proxy client, see
  xref:upgrade:proxy-intro.adoc[].
. After the proxy upgrade is complete, apply the highstate on the {smr} Branch
  Server.  When applying the highstate, the retail functionality will also be
  updated.
