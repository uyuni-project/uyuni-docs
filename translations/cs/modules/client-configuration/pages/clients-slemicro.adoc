[[clients-slemicro]]
= Registering {slea}{nbsp}{micro} Clients

This section contains information about registering clients running these {slea}{nbsp}{micro} operating systems:

* {slea}{nbsp}{micro}{nbsp}5.0 {x86_64}
* {slea}{nbsp}{micro}{nbsp}5.0 {arm64}

[WARNING]
====
Support for {slea}{nbsp}{micro} clients is provided as a technology preview for testing purposes, and not all features are fully functional at this stage. This feature is expected to be fully supported in a later version of {productname}. Do not use this feature on production systems.
====
