[[supported-features-sles]]
= Supported {slsa} and {leap} Features
:revdate: 2024-05-10
:page-revdate: {revdate}


This table lists the availability of various features on {suse} and {leap} clients. This table covers all variants of the {sle} operating system, including {slsa}, {sleda}, {sles} for SAP, and {sles} for HPC.

[IMPORTANT]
====
The operating system you run on a client is supported by the organization that supplies the operating system. {sle} is supported by {suse}. {leap} is supported by the {opensuse} community.
====

The icons in this table indicate:

* {check} the feature is available
* {cross} the feature is not available
* {question} the feature is under consideration, and may or may not be made available at a later date
