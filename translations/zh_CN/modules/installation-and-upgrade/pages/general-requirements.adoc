= 一般要求
:revdate: 2025-02-20
:page-revdate: {revdate}
ifeval::[{uyuni-content} == true]

:noindex:

在开始安装之前，请确保：

. {scclongform} 帐户。您可以使用此帐户访问 {productname} 服务器、代理和零售分支服务器的组织身份凭证和注册密钥。
. {productname} {webui} 支持的浏览器。
. 环境的 SSL 证书。默认情况下，{productname} {productnumber} 使用自我签名证书。

[NOTE]
====
{sl-micro} {microversion} 权利包含在 {productname} 权利中，因此不需要单独的注册密钥。
====

以下小节包含有关这些要求的详细信息。


[[install.scc-register]]
== {scclongform} 帐户和身份凭证


在部署 {productname} {productnumber} 之前，在 {scclongform} 中创建一个帐户。

[[creating.scc.account.mgr]]
.过程：获取组织身份凭证
. 在网页浏览器中导航到 link:https://scc.suse.com/login[]。
. 登录到您的 {scc} 帐户，或按照提示创建一个新帐户。
. 如果您尚未这样做，请单击 btn:[连接到组织] 并键入或搜索您的组织。
. 单击 btn:[管理我的组织]，然后在列表中单击您的组织名称以将其选中。
. 单击 btn:[用户] 选项卡，然后选择 btn:[组织身份凭证] 子选项卡。
. 记下您的登录信息，以便在设置 {productname} 期间使用。

根据组织的设置，还可能需要使用左侧导航栏中的 btn:[激活订阅] 菜单激活您的订阅。

有关使用 {scc} 的详细信息，请参见 https://scc.suse.com/docs/help[]。



[[installation-general-supportedbrowsers]]
== {productname} {webui} 支持的浏览器

要使用 {webui} 管理您的 {productname} 环境，必须运行最新的 Web 浏览器。

以下浏览器支持 {productname}：

* {sles} 随附的最新 Firefox 浏览器
* 所有操作系统上的最新 Chrome 浏览器
* Windows 随附的最新 Edge 浏览器

不支持 Windows Internet Explorer。在 Windows Internet Explorer 中无法正常呈现 {productname} {webui}。



== SSL 证书

{productname} 使用 SSL 证书来确保客户端注册到正确的服务器。{productname} 默认使用自我签名证书。如果您的证书已由第三方 CA 签名，可将其导入 {productname} 安装。

* 有关自我签名证书的详细信息，请参见 xref:administration:ssl-certs-selfsigned.adoc[]。
* 有关导入的证书的详细信息，请参见 xref:administration:ssl-certs-imported.adoc[]。
