.使用第三方 SSL 证书升级
[IMPORTANT]
====
如果您使用第三方证书，数据库容器需配备包含以下主体备用名称 (SAN) 的 SSL 证书：

* [literal]``db``
* [literal]``reportdb``
* 对外提供服务的完全限定域名

同一证书可同时用于主容器与数据库容器，但需包含上述 SAN。

In order to pass the new certificate to the upgrade command, use the [command]``--ssl-db-ca-root``, [command]``--ssl-db-cert`` and [command]``--ssl-db-key`` parameters.
====

.升级到特定版本
[NOTE]
====
如果未指定标记参数，则默认会升级到最新版本。要升级到特定版本，请为标记参数提供所需的映像标记。
====
