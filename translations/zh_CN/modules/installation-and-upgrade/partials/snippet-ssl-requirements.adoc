后续流程需要 SSL 证书。如果不使用自我签名的 CA 和证书，请在开始前准备好以下内容：

* 证书颁发机构 (CA) SSL 公共证书。如果您使用 CA 链，则所有中间 CA 也必须可用。
* SSL 数据库私用密钥。
* SSL 数据库证书。

所有文件必须采用 PEM 格式。

SSL 服务器证书的主机名必须与部署证书的计算机的完全限定主机名匹配。您可以在证书的 [literal]``X509v3 Subject Alternative Name`` 部分中设置主机名，也可以根据环境的需要列出多个主机名。支持的密钥类型为 [literal]``RSA`` 和 [literal]``EC``（椭圆曲线）。

[IMPORTANT]
====
数据库 SSL 证书的 [literal]``Subject Alternative Name`` 必须是 [literal]``reportdb``、[literal]``db`` 及用于访问报告数据库的 FQDN。
====
