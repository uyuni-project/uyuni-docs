== 为代理创建激活密钥


.过程：创建激活密钥
[role=procedure]
_____

. 导航到menu:系统[激活密钥]，然后单击 btn:[创建密钥]。

. 为代理主机创建激活密钥，并使用 {sl-micro} {microversion} 或 {sles} {bci-mlm} 作为父通道。该密钥应包含所有建议的通道，并包含代理作为扩展子通道。

. 继续将代理主机作为 [systemitem]``default`` 客户端进行引导。

_____

[WARNING]
====
Ensure the Proxy is assigned only to original vendor channels.

Assigning cloned channels is not supported at this moment.
====
