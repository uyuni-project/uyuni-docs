. 部署 {productname}。

+

[NOTE]
====
如果您使用 VM 映像作为迁移目标，最后一步请执行命令 [command]``mgradm migrate``，而非 [command]``mgradm install``。
====

+

根据 SSL 证书类型（自我签名证书或第三方证书），执行以下命令之一。
  请将 `<FQDN>` 替换为 {productname} 服务器的完全限定域名：

+

--
* 使用 {productname} 提供的自我签名证书：

+

[source, shell]
----
mgradm install podman <FQDN>
----

* 使用第三方 SSL 证书标志导入相应 SSL 证书（如果无需全部证书，可调整示例）：

+

[source, shell]
----
mgradm install podman <FQDN> \
  --ssl-ca-intermediate <strings> \
  --ssl-ca-root <string> \
  --ssl-server-cert <string> \
  --ssl-server-key <string> \
  --ssl-db-ca-intermediate <strings> \
  --ssl-db-ca-root <string> \
  --ssl-db-cert <string> \
  --ssl-db-key <string>
----

+

有关详细信息，请执行命令 [command]``mgradm install podman --help`` 查看帮助文档。

--
