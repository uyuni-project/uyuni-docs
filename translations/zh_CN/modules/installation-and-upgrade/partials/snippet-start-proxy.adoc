== 启动 {productname} {productnumber} 代理


现在可以使用 [literal]`mgrpxy` 命令启动容器：

.过程：启动代理并检查状态

. 调用以下命令启动代理：
+

----
mgrpxy start
----

. 调用以下命令检查容器状态：
+

----
mgrpxy status
----
+

应该会显示以下五个 {productname} 代理容器，并且它们应该是 [literal]``proxy-pod`` 容器 Pod 的一部分：

- proxy-salt-broker
- proxy-httpd
- proxy-tftpd
- proxy-squid
- proxy-ssh
