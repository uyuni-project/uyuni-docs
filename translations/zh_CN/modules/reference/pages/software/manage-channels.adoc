[[ref-software-manage-channels]]
= Manage Channels
:revdate: 2020-09-11
:page-revdate: {revdate}

The menu:Software[Manage > Channels] section allows you to manage custom channels.

Click btn:[Create Channel] to create a new custom channel.

To clone an existing channel, click btn:[Clone Channel] and select the channel to clone. You can choose to clone channel with or without all current patches, or select specific patches for inclusion.

有关自定义通道的详细信息，请参见 xref:administration:custom-channels.adoc[]。
