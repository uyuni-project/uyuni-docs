某些 {alibabaclo}{nbsp}2 实例在首次尝试注册时会失败。这由 {alibabaclo}{nbsp}2 映像中的一个已知 Bug 所导致。

`python-urlgrabber3` 软件包以 Python pip 软件包和 RPM 软件包两种形式提供，这可能会导致在首次尝试注册时发生冲突。

如果您的实例基于其中一个受影响的映像版本，客户端在第二次尝试注册时应该会正确注册。
