[[redhat-registration-overview]]
= Red Hat 客户端注册
:revdate: 2024-01-31
:page-revdate: {revdate}

您可以使用 {redhat} 内容分发网络 (CDN) 或 {redhat} 更新基础结构 (RHUI) 将 {rhel} 客户端注册到 {productname} 服务器。方法和细节视客户端的操作系统而异。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。

* 有关创建激活密钥的详细信息，请参见 xref:client-configuration:activation-keys.adoc[]。
* 有关将 {rhel} 迁移到 {sll} 的详细信息，请参见 xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]。
