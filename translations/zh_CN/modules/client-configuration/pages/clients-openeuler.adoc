[WARNING]
====
如果 {openeuler} 实例是在 AWS 中创建的，则其在 `/etc/machine-id` 中的 `machine-id` ID 始终相同。请务必在创建实例后重新生成 `machine-id`。有关详细信息，请参见 xref:administration:troubleshooting/tshoot-registerclones.adoc[]。
====



== 添加软件通道


将 {openeuler} 客户端注册到 {productname} 服务器之前，需要添加并同步所需的软件通道。

目前支持的体系结构有 `x86_64` 和 `aarch64`。
