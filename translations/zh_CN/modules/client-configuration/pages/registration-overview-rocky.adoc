[[rocky-registration-overview]]
= {rocky} 客户端注册
:revdate: 2024-01-31
:page-revdate: {revdate}

您可以将 {rocky}} 客户端注册到 {productname} 服务器。方法和细节视客户端的操作系统而异。

开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您必须已创建激活密钥。

* 有关创建激活密钥的详细信息，请参见 xref:client-configuration:activation-keys.adoc[]。
* 有关将 {rocky} 迁移到 {sll} 的详细信息，请参见 xref:client-configuration:clients-sleses.adoc#clients-sleses-el-migration[]。
