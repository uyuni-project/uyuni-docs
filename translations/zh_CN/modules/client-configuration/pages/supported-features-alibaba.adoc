= 支持的 {alibabaclo} 功能
:revdate: 2024-01-31
:page-revdate: {revdate}


下表列出了 {alibabaclo} 客户端上各种功能的可用性。

[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{alibabaclo} 由 Alibaba Cloud 支持。
====

此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
