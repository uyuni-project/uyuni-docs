== 前期步骤或检查


开始前，请确保客户端的日期和时间已与 {productname} 服务器正确同步。

您还必须已创建好激活密钥。有关创建激活密钥的详细信息，请参见 xref:client-configuration:activation-keys.adoc[]。



[IMPORTANT]
====
请不要将 {productname} 服务器注册到其自身。必须单独管理 {productname} 服务器，或者使用另一个独立的 {productname} 服务器来管理它。有关使用多个服务器的详细信息，请参见 xref:specialized-guides:large-deployments/hub-multi-server.adoc[]。
====



[[suse-registration-overview-check]]
== 检查之前的注册状态

如果客户端之前已注册到 SCC、SMT 或 RMT，请在将客户端迁移到 {productname} 服务器之前使用 [command]``SUSEConnect`` 去除此类注册。请务必先去除旧注册，否则，在使用 {productname} 将客户端迁移到更新版本的 {suse} 操作系统后，不再需要的储存库将添加到客户端。

要取消注册 {sle} 12 或 15 客户端，请在客户端上以 root 身份执行以下命令：

----
SUSEConnect --de-register
SUSEConnect --cleanup
----

有关具体的清理命令和校验步骤的详细信息，请参见 https://www.suse.com/de-de/support/kb/doc/?id=000019054[]。
