[[supported-features-sle-micro]]
= 支持的 {sle-micro} 功能
:revdate: 2024-07-18
:page-revdate: {revdate}


[IMPORTANT]
====
客户端上运行的操作系统由提供操作系统的组织支持。{sle-micro} 由 {suse} 支持。
====

[IMPORTANT]
====
目前，{sle-micro} 仅支持作为常规的受控端（[systemitem]``默认``联系方法）。我们正力求让其也能作为 Salt SSH 客户端（[systemitem]``salt-ssh`` 联系方法）进行管理。
====

此表中图标的含意如下：

* {check} 该功能可用
* {cross} 该功能不可用
* {question} 该功能正在考虑之中，日后可能提供，也可能不提供
