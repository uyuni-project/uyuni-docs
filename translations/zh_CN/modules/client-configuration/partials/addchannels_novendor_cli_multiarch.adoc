.过程：在命令提示符处添加软件通道
. 通过 [command]``spacewalk-common-channels`` 命令，您可添加对应的通道。以 root 用户身份在 {productname} 容器主机的命令行中，执行以下命令。请务必指定正确的体系结构：

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels \
-a <architecture> \
<base_channel_name> \
<child_channel_name_1> \
<child_channel_name_2> \
... <child_channel_name_n>
----
. 如果xref:administration:custom-channels.adoc#_custom_channel_synchronization[自动同步]处于关闭状态，请在容器主机上执行以下命令来同步通道：

+

[source, shell]
----
mgrctl exec -- spacewalk-repo-sync -p <base_channel_label>-<architecture>
----

. 确保同步已完成，然后再继续操作。
