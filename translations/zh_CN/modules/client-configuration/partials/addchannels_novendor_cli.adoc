.过程：在命令提示符处添加软件通道

. 通过 [command]``spacewalk-common-channels`` 命令，您可添加对应的通道。以 root 用户身份在 {productname} 容器主机的命令行中，执行以下命令：

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels \
<base_channel_label> \
<child_channel_label_1> \
<child_channel_label_2> \
... <child_channel_label_n>
----

+

要列出所有可用储存库，请执行以下命令：

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels -l
----

. 如果xref:administration:custom-channels.adoc#_custom_channel_synchronization[自动同步]处于关闭状态，请同步通道：

+

[source, shell]
----
mgrctl exec -- spacewalk-repo-sync -p <base_channel_label>
----

. 确保同步已完成，然后再继续操作。
