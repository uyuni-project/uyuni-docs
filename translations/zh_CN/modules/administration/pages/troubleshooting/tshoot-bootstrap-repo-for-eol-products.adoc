[[troubleshooting-bootstrap-repo-for-eol-products]]
= 对生命周期已结束产品的引导储存库进行查错
:revdate: 2024-10-31
:page-revdate: {revdate}

同步受支持的产品时，会自动在 {productname} Server 上创建及重新生成引导储存库。当产品到达生命周期结束日期且不再受支持时，如果您要继续使用此产品，就必须手动创建引导储存库。

有关引导储存库的详细信息，请参见 xref:client-configuration:bootstrap-repository.adoc[]。



.过程：创建生命周期已结束产品的引导储存库

. 在 {productname} 容器主机的命令提示符处，以 root 身份进入服务器容器：
+

----
mgrctl term
----

. 在容器内部，执行以下步骤：
+

--
.. 使用 [option]``--force`` 选项列出可用的不受支持的引导储存库，例如：
+

----
mgr-create-bootstrap-repo --list --force
1. SLE-12-SP2-x86_64
2. SLE-12-SP3-x86_64
----

.. 使用适当的储存库名称作为产品标签，创建引导储存库：
+

----
mgr-create-bootstrap-repo --create SLE-12-SP2-x86_64 --force
----
--

如果您不想手动创建引导储存库，可以检查您需要的产品和引导储存库是否有 LTSS。
