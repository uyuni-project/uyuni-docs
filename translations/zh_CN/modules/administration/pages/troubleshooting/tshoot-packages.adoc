////
PUT THIS COMMENT AT THE TOP OF TROUBLESHOOTING SECTIONS

Troubleshooting format:

One sentence each:
Cause: What created the problem?
Consequence: What does the user see when this happens?
Fix: What can the user do to fix this problem?
Result: What happens after the user has completed the fix?

If more detailed instructions are required, put them in a "Resolving" procedure:
.Procedure: Resolving Widget Wobbles
. First step
. Another step
. Last step
////

当客户端上的软件包被锁定时，{productname} Server 可能无法正确确定适用的补丁集。如果发生这种情况，软件包更新会显示在 {webui} 中，但不会显示在客户端上，并且尝试更新客户端会失败。检查软件包锁定和排除列表，以确定是否在客户端上锁定或排除了软件包。

在客户端上，检查软件包锁定和排除列表，以确定是否已锁定或排除软件包：

// SUSE Liberty Linux not available at Uyuni for now

ifeval::[{mlm-content} == true]

* 在扩展支持平台上，检查 [path]``/etc/yum.conf`` 并搜索 ``exclude=``。
endif::[]

* 在 {sle} 和 {opensuse} 上，使用 [command]``zypper locks`` 命令。
