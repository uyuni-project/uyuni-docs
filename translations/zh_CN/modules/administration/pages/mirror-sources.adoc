[[security-morror-sources]]
= 镜像源软件包
:revdate: 2024-05-10
:page-revdate: {revdate}

如果您在本地构建自己的软件包，或者出于法律原因需要提供软件包的源代码，可以在 {productname} Server 上镜像源软件包。

[NOTE]
====
镜像源软件包可能会消耗大量磁盘空间。
====

[IMPORTANT]
====
请在服务器容器内执行相应步骤之前使用 [literal]``mgrctl term``。
====

.过程：镜像源软件包
. 打开 [filename]``/etc/rhn/rhn.conf`` 配置文件并添加下面一行内容：
+

----
server.sync_source_packages = 1
----
. 重启 Spacewalk 服务以应用更改：
+

----
mgradm restart
----

目前，只能为所有储存库全局启用此功能。无法选择单个要镜像的储存库。

激活此功能后，源软件包会在下次储存库同步后显示在 {productname} {webui} 中。它们显示为二进制软件包的源，可以直接从 {webui} 下载。无法使用 {webui} 在客户端上安装源软件包。
