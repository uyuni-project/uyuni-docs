== 使用 Ansible 剧本进行修复


需要一个 Ansible 控制节点。有关详细信息，请参见 xref:administration:ansible-setup-control-node.adoc[]。

以下过程将介绍如何使用 Ansible 剧本运行修复任务。

.过程：使用 Ansible 剧本运行修复

. 在控制节点系统菜单中选择 menu:Ansible[剧本]，然后单击某个剧本，例如：

+

[source, shell]
----
sle15-playbook-stig.yml
----

. 要运行该剧本，请选择客户端的[guimenu]``清单路径``，例如：

+

[source, shell]
----
/etc/ansible/sles15
----

+

单击 btn:[日程安排]。

. 在[guimenu]``事件``选项卡下检查已安排事件的状态。



如果剧本位于其他目录下，您可以单击[guimenu]``设置 Ansible 控制节点``链接了解如何添加该节点。
