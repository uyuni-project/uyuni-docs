.プロシージャ: コマンドプロンプトからのソフトウェアチャンネルの追加
. [command]``spacewalk-common-channels``コマンドを使用して、適切なチャンネルを追加できます。 {productname}コンテナホストのコマンドプロンプトで、rootとして次のコマンドを実行します。 このとき、正しいアーキテクチャを指定してください。

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels \
-a <architecture> \
<base_channel_name> \
<child_channel_name_1> \
<child_channel_name_2> \
... <child_channel_name_n>
----
. xref:administration:custom-channels.adoc#_custom_channel_synchronization[自動同期]がオフになっている場合は、次のコマンドを使用してコンテナホスト上のチャンネルを同期します。

+

[source, shell]
----
mgrctl exec -- spacewalk-repo-sync -p <base_channel_label>-<architecture>
----

. 続行前に、同期が完了していることを確認してください。
