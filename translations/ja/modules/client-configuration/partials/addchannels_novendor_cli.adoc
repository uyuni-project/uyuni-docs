.プロシージャ: コマンドプロンプトからのソフトウェアチャンネルの追加

. [command]``spacewalk-common-channels``コマンドを使用して、適切なチャンネルを追加できます。 {productname}コンテナホストのコマンドプロンプトで、rootとして次のコマンドを実行します。

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels \
<base_channel_label> \
<child_channel_label_1> \
<child_channel_label_2> \
... <child_channel_label_n>
----

+

使用可能なすべてのリポジトリを一覧にするには、次のコマンドを実行します。

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels -l
----

. xref:administration:custom-channels.adoc#_custom_channel_synchronization[自動同期]がオフになっている場合は、チャンネルを同期します。

+

[source, shell]
----
mgrctl exec -- spacewalk-repo-sync -p <base_channel_label>
----

. 続行前に、同期が完了していることを確認してください。
