= コンテナの起動と停止
:revdate: 2024-02-13
:page-revdate: {revdate}

次のコマンドを使用して、{productname} {productnumber}サーバコンテナを再起動、起動、および停止できます。


{productname} {productnumber}サーバを再起動([command]``restart``)するには、次のコマンドを実行します。
----
# mgradm restart
5:23PM INF Welcome to mgradm
5:23PM INF Executing command: restart
----

サーバを起動([command]``start``)するには、次のコマンドを実行します。
----
# mgradm start
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: start
----

サーバを停止([command]``stop``)するには、次のコマンドを実行します。
----
# mgradm stop
5:21PM INF Welcome to mgradm
5:21PM INF Executing command: stop
----
