[[persistant-volume-list]]
= 永続コンテナボリューム
:revdate: 2025-07-29
:page-revdate: {revdate}

コンテナ内で行った変更は保持されません。永続ボリュームの外部で加えた変更は破棄されます。以下に{productname} {productnumber}の永続ボリュームのリストを示します。

デフォルトのボリュームの場所をカスタマイズするには、[command]``podman volume create``コマンドを使用して、ポッドの最初の起動前に、必要なボリュームを作成するようにします。

[NOTE]
====
この表は、Helmチャートおよびsystemctlサービス定義の両方で示されているボリュームマッピングに正確に従っています。
====


== サーバ


次のボリュームは、サーバ上の**Podman**のデフォルトのストレージの場所に保存されます。

[cols="name,directory"]
|===
| ボリューム名 | ボリュームディレクトリ

| **Podman Storage**
| [path]``/var/lib/containers/storage/volumes/``
|===
