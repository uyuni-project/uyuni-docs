== プロキシのアクティベーションキーの作成


.プロシージャ: アクティベーションキーの作成
[role=procedure]
_____

. menu:システム[アクティベーションキー]に移動し、btn:[キーの作成]をクリックします。

. {sl-micro} {microversion}または{sles} {bci-mlm}を親チャンネルとして使用して、プロキシホストのアクティベーションキーを作成します。このキーには、推奨されるすべてのチャンネルと、拡張機能としてのプロキシの子チャンネルが含まれる必要があります。

. 続行して、プロキシホストを[systemitem]``デフォルト``クライアントとしてブートストラップします。

_____

[WARNING]
====
Ensure the Proxy is assigned only to original vendor channels.

Assigning cloned channels is not supported at this moment.
====
