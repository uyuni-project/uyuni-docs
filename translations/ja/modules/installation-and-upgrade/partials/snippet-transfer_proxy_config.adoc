== プロキシ設定の転送


{webui}によって設定アーカイブが生成されます。このアーカイブをプロキシコンテナホストで利用できるようにする必要があります。

// On sl-micro, root login with password req
// FIXME: Adding link to end of prepare-micro-host.adoc?

.プロシージャ: プロキシ設定をコピーする
[role=procedure]
_____

. まだ実行していない場合は、前のステップで生成された設定アーカイブ ([literal]``config.tar.gz``)をサーバコンテナからサーバホストにコピーします。

+

----
mgrctl cp server:/root/config.tar.gz .
----

. まだ実行していない場合は、サーバホストからプロキシホストにファイルをコピーします。

+
----
scp config.tar.gz <proxy-FQDN>:/root
----

_____
