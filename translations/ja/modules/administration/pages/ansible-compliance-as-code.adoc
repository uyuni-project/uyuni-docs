== Ansible Playbookを使用した修復


Ansible制御ノードが必要です。 詳細については、xref:administration:ansible-setup-control-node.adoc[]を参照してください。

次のプロシージャでは、Ansible playbookを使用して修復を実行する方法について説明します。

.プロシージャ: Ansible Playbookを使用して修復を実行する

. 制御ノードのシステムメニューから、menu:Ansible[playbook]を選択し、playbookをクリックします。例:

+

[source, shell]
----
sle15-playbook-stig.yml
----

. playbookを実行するには、クライアントの［[guimenu]``インベントリパス``］を選択します。例:

+

[source, shell]
----
/etc/ansible/sles15
----

+

btn:[Schedule] をクリックします。

. ［ [guimenu]``イベント``］タブで、スケジュールされたイベントのステータスを確認します。



playbookが別のディレクトリにある場合は、［[guimenu]``Ansible制御ノードのセットアップ``］へのリンクをたどって、その追加方法を確認できます。
