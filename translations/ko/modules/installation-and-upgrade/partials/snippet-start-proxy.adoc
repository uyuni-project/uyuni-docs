== {productname} {productnumber} 프록시 시작


이제 [literal]`mgrpxy` 명령으로 컨테이너를 시작할 수 있습니다.

.절차: 프록시 시작 및 상태 확인

. 다음을 호출하여 프록시 시작:
+

----
mgrpxy start
----

. 다음을 호출하여 컨테이너 상태 확인:
+

----
mgrpxy status
----
+

5개의 {productname} 프록시 컨테이너가 있어야 하며, 이러한 컨테이너는 [literal]``proxy-pod`` 컨테이너 Pod에 속해야 합니다.

- proxy-salt-broker
- proxy-httpd
- proxy-tftpd
- proxy-squid
- proxy-ssh
