== 프록시 호스트를 클라이언트로 부트스트랩


// On sl-micro, root login with password req
// FIXME: Adding link to end of prepare-micro-host.adoc?

.Procedure: Bootstrapping the Proxy Host
[role=procedure]
_____

. menu:시스템[부트스트랩]을 선택합니다.

. 프록시 호스트의 필드를 입력합니다.

. 드롭다운에서 이전 단계에서 생성한 활성화 키를 선택합니다.

. btn:[부트스트랩]을 클릭합니다.

. 부트스트랩 프로세스가 완료될 때까지 기다립니다. menu:Salt[] 메뉴를 선택한 후 {salt} 키가 나열되고 수락되었는지 확인합니다.

. 운영 체제가 {sl-micro}인 경우 프록시 호스트를 재부팅합니다.

. menu:시스템[] 목록에서 호스트를 선택하고 모든 이벤트가 완료된 후 {sl-micro}의 경우 두 번째 재부팅을 트리거하여 온보딩을 완료합니다.

_____


.Procedure: Updating the Proxy Host
[role=procedure]
_____

. menu:시스템[] 목록에서 호스트를 선택하고 모든 패치를 적용하여 업데이트합니다.

. 운영 체제가 {sl-micro}인 경우 프록시 호스트를 재부팅합니다.

_____
