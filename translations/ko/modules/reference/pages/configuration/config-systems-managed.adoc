= Managed Systems
:revdate: 2020-10-27
:page-revdate: {revdate}

By default the [guimenu]``Managed Systems`` page is displayed. The listed systems have been fully prepared for configuration file deployment. The page shows systems with managed configuration files and channel subscriptions.

Click the name of a system to show the relevant menu:System Details[Configuration > Overview] page.
