== Ansible 플레이북을 사용한 수정


Ansible 제어 노드가 필요합니다. 자세한 내용은 xref:administration:ansible-setup-control-node.adoc[]에서 확인할 수 있습니다.

다음 절차는 Ansible 플레이북을 사용하여 교정을 실행하는 과정을 안내합니다.

.절차: Ansible 플레이북을 사용한 수정 실행

. 제어 노드 시스템 메뉴에서 menu:Ansible[플레이북]을 선택하고 플레이북을 클릭합니다. 예는 다음과 같습니다.

+

[source, shell]
----
sle15-playbook-stig.yml
----

. 플레이북을 실행하려면 클라이언트의 [guimenu]``인벤토리 경로``를 선택합니다. 예는 다음과 같습니다.

+

[source, shell]
----
/etc/ansible/sles15
----

+

btn:[일정]을 클릭합니다.

. [guimenu]``이벤트`` 탭에서 예약된 이벤트의 상태를 확인합니다.



플레이북이 다른 디렉토리에 있는 경우, [guimenu]``Ansible 제어 노드 설정`` 링크를 따라 추가하는 방법을 확인할 수 있습니다.
