.절차: 명령 프롬프트에서 소프트웨어 채널 추가

. [command]``spacewalk-common-channels`` 명령을 사용하여 적절한 채널을 추가할 수 있습니다. {productname} 컨테이너 호스트의 명령 프롬프트에서 루트 권한으로 다음 명령을 실행합니다.

+

[source, shell]
----
mgrctl exec -ti -- spacewalk-common-channels \
<base_channel_label> \
<child_channel_label_1> \
<child_channel_label_2> \
... <child_channel_label_n>
----

+

사용 가능한 모든 리포지토리를 나열하려면 다음 명령을 실행합니다.

+

[source, shell]
----
mgrctl exec -ti -- spacewalk-common-channels -l
----

. xref:administration:custom-channels.adoc#_custom_channel_synchronization[automatic synchronization]이 꺼진 경우 다음 채널을 동기화하십시오.

+

[source, shell]
----
mgrctl exec -ti -- spacewalk-repo-sync -p <base_channel_label>
----

. 계속하기 전에 동기화가 완료되었는지 확인합니다.
