.절차: 명령 프롬프트에서 소프트웨어 채널 추가
. [command]``spacewalk-common-channels`` 명령을 사용하여 적절한 채널을 추가할 수 있습니다. {productname} 컨테이너 호스트의 명령 프롬프트에서 루트 권한으로 다음 명령어를 실행합니다. 올바른 아키텍처를 지정했는지 확인합니다.

+

[source, shell]
----
mgrctl exec -- spacewalk-common-channels \
-a <architecture> \
<base_channel_name> \
<child_channel_name_1> \
<child_channel_name_2> \
... <child_channel_name_n>
----
. xref:administration:custom-channels.adoc#_custom_channel_synchronization[자동 동기화]가 비활성 상태인 경우 다음 명령으로 컨테이너 호스트의 채널을 동기화합니다.

+

[source, shell]
----
mgrctl exec -- spacewalk-repo-sync -p <base_channel_label>-<architecture>
----

. 계속하기 전에 동기화가 완료되었는지 확인합니다.
