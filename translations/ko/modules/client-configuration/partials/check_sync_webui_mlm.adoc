.Procedure: Checking synchronization progress from the {webui}
[role=procedure]
____
. {productname} {webui}에서 menu:관리[설치 마법사]로 이동하여 [guimenu]``제품`` 탭을 선택합니다. 이 대화 상자는 동기화되는 중에 각 제품에 대한 진행률 막대를 표시합니다.

. 또는 menu:소프트웨어[관리 > 채널]로 이동한 다음, 리포지토리에 연결된 채널을 클릭하면 됩니다. [guimenu]``리포지토리`` 탭으로 이동한 다음, [guimenu]``동기화``를 클릭하고 [systemitem]``동기화 상태``를 선택합니다.
____
