[[supported-features-sle-micro]]
= 지원되는 {sle-micro} 기능
:revdate: 2024-07-18
:page-revdate: {revdate}


[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {suse}는 {sle-micro}를 지원합니다.
====

[IMPORTANT]
====
{sle-micro}는 당분간 일반 미니언([systemitem]``기본`` 연락 방법)으로만 지원됩니다. Salt SSH 클라이언트([systemitem]``salt-ssh`` 연락 방법)로도 관리하는 작업이 진행 중입니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능을 사용할 수 있음
* {cross} 이 기능을 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후에 사용할 수도 있고 사용하지 못할 수도 있습니다.
