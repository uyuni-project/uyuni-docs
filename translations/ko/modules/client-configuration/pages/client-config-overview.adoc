= 클라이언트 구성 안내서
:revdate: 2025-05-21
:page-revdate: {revdate}

// HTML Publication date 
**게시 날짜:** {docdate}

== 서문


{productname} 설치 이후의 첫 번째 단계는 클라이언트 등록입니다. 이 클라이언트 관리는 워크플로우의 핵심이 됩니다.

{productname}은(는) {sle} 또는 다른 Linux 배포판을 지원하며, 다양한 하드웨어 옵션에 걸쳐 호환성을 제공합니다.

지원되는 클라이언트 및 기능의 전체 목록은 xref:client-configuration:supported-features.adoc[지원되는 클라이언트 및 기능]에서 확인할 수 있습니다.

이 안내서는 클라이언트의 수동 및 자동 등록과 구성을 모두 다룹니다.
