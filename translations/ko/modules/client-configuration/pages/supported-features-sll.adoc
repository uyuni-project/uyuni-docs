[[supported-features-sll]]
= 지원되는 {sll} 기능
:revdate: 2024-09-30
:page-revdate: {revdate}

아래 표는 {sll} 클라이언트에서 다양한 기능의 사용 가능 여부를 정리한 것입니다.

[NOTE]
====
{sll} 클라이언트를 {sleses}(SLES ES), Liberty, RES 또는 {redhat} {es}라고도 합니다.
====

[IMPORTANT]
====
클라이언트에서 실행하는 운영 체제는 운영 체제를 제공하는 조직이 지원합니다. {suse}는 {sll}을 지원합니다.
====

이 표의 아이콘에는 다음과 같은 의미가 있습니다.

* {check} 이 기능을 사용할 수 있음
* {cross} 이 기능을 사용할 수 없습니다.
* {question} 이 기능은 현재 고려 중이며, 이후에 사용할 수도 있고 사용하지 못할 수도 있습니다.
