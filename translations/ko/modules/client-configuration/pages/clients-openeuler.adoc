이 섹션에는 {openeuler} 운영 체제를 실행하는 클라이언트를 등록하는 방법에 대한 정보가 포함되어 있습니다.

ifeval::[{mlm-content} == true]

// Use list format: repository URLs / packages and metadata / version info
[IMPORTANT]
====
* {openeuler} 리포지토리 URL은 {scclongform}에서 사용할 수 있습니다.
* 패키지 및 메타데이터는 {suse}가 아닌 OpenAtom Foundation이 제공합니다.
* 지원되는 제품에 대해서는 지원 테이블 및 릴리스 노트를 참조하십시오.
====
endif::[]


[WARNING]
====
AWS에서 생성되었을 때 {openeuler} 인스턴스는 항상 ``/etc/machine-id``에서 동일한 ``machine-id`` ID를 갖습니다. 인스턴스가 생성된 후 ``machine-id``를 다시 생성해야 합니다. 자세한 내용은 xref:administration:troubleshooting/tshoot-registerclones.adoc[]에서 확인할 수 있습니다.
====



== 소프트웨어 채널 추가

현재 지원되는 아키텍처: ``x86_64`` 및 ``aarch64``.
