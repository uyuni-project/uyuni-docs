密钥

The Salt  Keys section displays the key fingerprints of your current Salt clients.

Key fingerprints are exchanged between the Uyuni Server and Salt clients to verify the identity of the server and the client. This prevents Salt clients from connecting to the wrong server.

Click Refresh to update the list. Click the name of a client to go to Systems  Details for that client.

Table 1. Salt Keys List Columns
Column Description

Name

Name of the Salt client.

Fingerprint

Key fingerprint of the Salt client.

State

The status of the key exchange: accepted indicates that the client key has been verified by the Uyuni Server.

Actions

Click the Delete icon to delete the client key from the server. Clients that have had their key deleted will need to be onboarded again.
